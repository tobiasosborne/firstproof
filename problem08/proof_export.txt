## Node 1

**Statement:** A polyhedral Lagrangian surface K in R^4 with exactly 4 faces meeting at every vertex necessarily has a Lagrangian smoothing, i.e., there exists a Hamiltonian isotopy K_t of smooth Lagrangian submanifolds parametrized by (0,1] extending to a topological isotopy on [0,1] with K_0 = K.

**Type:** claim

**Inference:** assumption

**Status:** pending

**Taint:** unresolved

### Node 1.1

**Statement:** Answer: YES. K necessarily has a Lagrangian smoothing.

PROOF STRATEGY. The proof proceeds in seven major steps:

STEP 1 (VERTEX CLASSIFICATION). Node 1.2 (VALIDATED) classifies all vertices. Every vertex where exactly 4 faces meet in a polyhedral Lagrangian surface is Type A: the 4 Lagrangian planes Pi_1, Pi_2, Pi_3, Pi_4 satisfy (i) each pair of consecutive planes shares a line (edge), (ii) each pair of non-consecutive planes is transverse. Moreover, all Type A vertices are Sp(4,R)-equivalent to a standard normal form depending on a single real parameter gamma in R. By reduction, gamma can be eliminated (all vertices equivalent to gamma=0). No Type B vertices (opposite half-planes coplanar) can occur at vertices of a topological surface.

STEP 2 (LOCAL VERTEX SMOOTHING). Node 1.3 (PENDING) constructs a smooth Lagrangian disk L_v in each vertex ball B_delta(v), generated by a compactly supported Hamiltonian isotopy. The construction uses cotangent generating functions with a two-zone design: (i) Inner zone (|X| < epsilon): F_smooth = 0, giving a flat Lagrangian plane (the zero section of T*R^2). (ii) Outer zone (epsilon <= |X| <= delta): Angular interpolation F_angular using a smooth partition of unity on S^1, multiplied by a radial cutoff chi(|X|/epsilon) that is flat at |X| = epsilon. The generating function is globally C^infinity on the base disk, including at the origin (D^2F_smooth(0) = 0, a well-defined zero matrix). The Hamiltonian isotopy is a shrinking inner-zone family: start with epsilon = 0 (surface = K, piecewise linear) and expand to epsilon = delta/8 (surface = L_v, smooth), using a flat reparametrization tau(t) to ensure smooth extension of the Hamiltonian to t = 0. Compact support: supp(H_v) subset B_delta(v).

STEP 3 (EDGE SMOOTHING). Node 1.4 (VALIDATED) smooths V-shaped edge singularities via product Lagrangian construction. In edge-adapted coordinates (x_1, x_2, y_1, y_2) where the edge is the x_1-axis and the two adjacent Lagrangian faces lie in {y_1 = 0}, the V-shaped edge has profile gamma_0(s) = (x_2(s), y_2(s)) (two rays in the transverse plane). Replace gamma_0 with a smooth curve gamma_1 having the same endpoints at infinity. The smoothed edge is L_e = {(x_1, gamma_1(s), 0, gamma_2(s)) : (x_1,s) in R x R}, a product of R (along x_1) and the curve (gamma_1(s), 0, gamma_2(s)) in the (x_2, y_1, y_2)-space. This is automatically Lagrangian: omega pulls back to 0 because dy_1 = 0 identically and dx_2, dy_2 depend only on the single parameter s (iota*(dx_2 ^ dy_2) = 0). The profile is constant (x_1-independent), avoiding cross-term issues. The construction is generated by a Hamiltonian supported in a tubular neighborhood of the edge.

STEP 4 (GLOBAL ASSEMBLY). Node 1.5 (PENDING) combines vertex and edge smoothings into a smooth embedded Lagrangian surface L via a two-phase Hamiltonian isotopy. Phase A (vertices): For each vertex v_i, choose radius delta_i small enough that the vertex balls B_{delta_i}(v_i) are pairwise disjoint. Apply Node 1.3's construction simultaneously at all vertices. The combined Hamiltonian H_A = sum_i H_{v_i} has pairwise disjoint supports, so Poisson brackets vanish and the flows commute. Output: K_A = phi_1^{H_A}(K), smooth inside each vertex ball and equal to K outside. Phase B (edges): For each edge e_k, apply Node 1.4's construction to the portion of the edge outside all vertex balls (the edge exterior e_k^o). Choose tubular neighborhood radii epsilon_k such that the edge tubes are pairwise disjoint. The combined Hamiltonian H_B = sum_k H_{e_k} again has pairwise disjoint supports. Output: L = phi_1^{H_B}(K_A), smooth everywhere. Sequential composition: phi_t^{total} = phi_{2t}^{H_A} for t in [0,1/2], then phi_t^{total} = phi_{2t-1}^{H_B} o phi_1^{H_A} for t in [1/2,1]. Phases A and B are sequential (not simultaneous) because their supports overlap in transition zones near vertex ball boundaries. Compact support: supp(H_A) cup supp(H_B) is contained in an arbitrarily small neighborhood of the 1-skeleton of K.

Boundary matching (key technical point): At the junction between Phase A and Phase B (the boundary of vertex balls along edges), Phase A produces a smooth surface. Phase B acts on this already-smooth surface, extending the smoothing further along the edge. No mismatch occurs because Phase A's generating function construction extends slightly beyond the vertex ball to smooth the edge creases approaching the vertex, ensuring the surface is smooth at the junction radius. The edge-adapted coordinates from Node 1.4 are compatible with the cotangent graph coordinates from Node 1.3 in the overlap region (both describe the same Lagrangian surface using different local charts).

STEP 5 (HAMILTONIAN ISOTOPY VERIFICATION). Node 1.6 (PENDING) verifies that the isotopy K_t from K to L is Hamiltonian throughout. Phase A is Hamiltonian by construction (Node 1.3's generating function isotopy, with explicit time-dependent Hamiltonian H_t(X,Y) = (d/dt)G_t(X) where G_t is the family of generating functions). Phase B is Hamiltonian (Node 1.4's edge profile deformation, with H(x_1,x_2,y_1,y_2) constructed via the Lagrangian graph isotopy). Concatenation of Hamiltonian isotopies is Hamiltonian (using a smooth time reparametrization to join the two phases). The total flow phi_t^{total} for t in [0,1] is a Hamiltonian isotopy in the ambient symplectic manifold (R^4, omega_std).

STEP 6 (TOPOLOGICAL EXTENSION TO t=0). Node 1.7 (PENDING) extends the smooth Hamiltonian isotopy {K_t}_{t in (0,1]} to a continuous family on [0,1] with K_0 = K. The key argument is uniform convergence: as t -> 0, the smoothing parameters shrink (vertex inner zone radius epsilon(t) -> 0, edge smoothing profile amplitude -> 0), and the smooth surfaces K_t converge uniformly on compact sets to the piecewise-linear surface K. The convergence is in the C^0 topology (Hausdorff distance on compact subsets), not C^1 (the derivatives blow up at the creases). The limiting surface K_0 = lim_{t->0} K_t is the original polyhedral surface K. This gives a topological isotopy on [0,1].

STEP 7 (OBSTRUCTION ANALYSIS). Node 1.8 (PENDING) verifies that no global obstructions prevent the smoothing. (a) Maslov class: The Maslov class is a homotopy invariant, so Mas(L) = Mas(K). Polyhedral Lagrangians have well-defined Maslov class (computed from face normals), and the Lagrangian isotopy preserves this class. (b) Floer homology: R^4 is convex at infinity, so Floer theory is well-behaved. The Lagrangian smoothing does not introduce new topology (L and K are isotopic, hence diffeomorphic as abstract surfaces). (c) Topological constraints: The surface K is a topological submanifold (link at each vertex is a simple closed curve for Type A vertices), so smoothing is topologically possible. (d) Monodromy: The local normal form at each vertex is unique up to symplectomorphism, so there is no monodromy obstruction to patching local smoothings.

### Node 1.2

**Statement:** Local vertex model classification (Lemma). Inference type: lemma.

CONVENTIONS. Coordinates (x_1, x_2, y_1, y_2) on R^4, identified with standard basis vectors e_1 = (1,0,0,0), e_2 = (0,1,0,0), e_3 = (0,0,1,0), e_4 = (0,0,0,1). The standard symplectic form is omega = dx_1 ^ dy_1 + dx_2 ^ dy_2, equivalently omega(e_1, e_3) = 1, omega(e_2, e_4) = 1, and all other basis pairings vanish. A 2-plane Pi in R^4 is Lagrangian if omega|_Pi = 0. The Lagrangian Grassmannian LG(2,4) parametrizes all such planes.

WARNING ON COORDINATE CONVENTION. Under the alternative convention (x_1, y_1, x_2, y_2) with basis ordering e_1 = x_1, e_2 = y_1, e_3 = x_2, e_4 = y_2, the symplectic form pairs the first two basis vectors nontrivially: omega(e_1, e_2) = 1. In THAT convention, span{e_1, e_2} would NOT be Lagrangian. All formulas in this node and its dependents use the convention (x_1, x_2, y_1, y_2) stated above.

STATEMENT (Lemma 1.2). Let K be a polyhedral Lagrangian surface in (R^4, omega_std) that is a topological submanifold, with exactly 4 faces meeting at each vertex. Let v be any vertex of K. Then:

(a) TANGENT CONE STRUCTURE. The tangent cone TC_v(K) consists of 4 Lagrangian conic sectors S_1, S_2, S_3, S_4 (cyclically ordered). Each sector S_i is a 2-dimensional cone (bounded by two edge rays from v) contained in a Lagrangian plane Pi_i. The 4 edge rays are the pairwise intersections of consecutive planes: ell_{i,i+1} = Pi_i cap Pi_{i+1} (indices mod 4). Consecutive planes share exactly a line.

(b) TYPE B IS IMPOSSIBLE. All 4 Lagrangian planes Pi_1, Pi_2, Pi_3, Pi_4 are pairwise distinct, and non-consecutive planes are transverse: Pi_1 cap Pi_3 = {0} and Pi_2 cap Pi_4 = {0}. The "Type B" configuration (opposite sectors coplanar) cannot occur at vertices of a topological submanifold.

(c) TOPOLOGICAL DISK. The link of v in S^3 = partial B_epsilon(v) is a simple closed curve composed of 4 great circle arcs. The tangent cone TC_v(K) is homeomorphic to a closed disk D^2.

(d) Sp(4,R) NORMAL FORM. All Type A vertex configurations are Sp(4,R)-equivalent. After applying a symplectomorphism, the 4 Lagrangian planes take the standard form:
  Pi_1 = span{e_1, e_2},  Pi_2 = span{e_1, e_4},
  Pi_3 = span{e_3, e_4},  Pi_4 = span{e_2, e_3},
with edge rays ell_{12} = R*e_1, ell_{23} = R*e_4, ell_{34} = R*e_3, ell_{41} = R*e_2. The moduli space of Type A configurations under Sp(4,R) is a single point (0-dimensional).

Remark on the 1-parameter family. In a partial normalization fixing only Pi_1, Pi_2, ell_{12}, and ell_{41}, the configuration depends on a parameter gamma:
  Pi_3 = span{e_4, gamma*e_1 + e_3},  Pi_4 = span{e_2, gamma*e_1 + e_3}.
However, the symplectomorphism g defined by g(e_3) = e_3 - gamma*e_1 (identity on e_1, e_2, e_4) sends C(gamma) to C(0). So gamma is NOT an invariant; it is an artifact of the incomplete normalization. Node 1.3 may use the C(gamma) form for computational convenience (the explicit construction works for all gamma and the smoothing is equivariant under the symplectomorphism g), but all such configurations are equivalent.

(e) LAGRANGIAN GRASSMANNIAN. LG(2,4) is diffeomorphic to U(2)/O(2), a compact 3-manifold with pi_1(LG(2,4)) = Z.

PROOF.

STEP 1 (TANGENT CONE STRUCTURE). Translate v to the origin. Since K is polyhedral with 4 faces meeting at v, the tangent cone TC_0(K) is the union of 4 conic sectors S_1, ..., S_4. Each S_i lies in the Lagrangian plane Pi_i tangent to the corresponding face at v (well-defined since each face is a flat Lagrangian polygon). Each S_i is bounded by two edge rays from the origin.

Consecutive faces F_i and F_{i+1} share an edge, so Pi_i and Pi_{i+1} share the direction along that edge: dim(Pi_i cap Pi_{i+1}) >= 1. Since Pi_i and Pi_{i+1} are tangent planes to distinct faces meeting along an edge (not coinciding), they are distinct 2-planes, so their intersection is exactly 1-dimensional: dim(Pi_i cap Pi_{i+1}) = 1 for all consecutive i (mod 4). The shared edge ray is ell_{i,i+1} = Pi_i cap Pi_{i+1}, and S_i is the cone in Pi_i bounded by ell_{i-1,i} and ell_{i,i+1}. This proves part (a).

STEP 2 (TYPE B IMPOSSIBLE). We prove dim(Pi_1 cap Pi_3) = 0 (and symmetrically dim(Pi_2 cap Pi_4) = 0).

Case dim(Pi_1 cap Pi_3) = 2: Then Pi_1 = Pi_3. The consecutive intersections ell_{12} = Pi_1 cap Pi_2 and ell_{23} = Pi_2 cap Pi_3 = Pi_2 cap Pi_1 are both equal to Pi_1 cap Pi_2, so ell_{12} = ell_{23}. Similarly ell_{34} = ell_{41}. Only 2 distinct edge rays exist. The tangent cone is the union of two planes sharing a line, whose link in S^3 is two great circles meeting at 2 points (a figure-eight). This is not S^1, contradicting the topological manifold hypothesis.

Case dim(Pi_1 cap Pi_3) = 1: Let ell = Pi_1 cap Pi_3 (a line). Since Pi_1 and Pi_3 are both Lagrangian and both contain ell, every u in ell satisfies omega(u, w) = 0 for all w in Pi_1 + Pi_3. Since dim(Pi_1 + Pi_3) = 2 + 2 - 1 = 3, we have ell = (Pi_1 + Pi_3)^omega (the 1-dimensional symplectic complement).

KEY CONSEQUENCE: All 4 edge rays lie in V := Pi_1 + Pi_3 (3-dimensional). Indeed, ell_{12}, ell_{41} lie in Pi_1 subset V and ell_{23}, ell_{34} lie in Pi_3 subset V.

We now show every Lagrangian plane contained in V must contain ell. Let Pi be Lagrangian with Pi subset V. The restriction omega|_V has radical rad(omega|_V) = V^omega = ell (1-dimensional). Project pi: V -> V/ell (2-dimensional, with nondegenerate induced symplectic form omega_bar). The image pi(Pi) is isotropic in (V/ell, omega_bar). Since V/ell is 2-dimensional symplectic, its maximal isotropic subspaces are 1-dimensional. Since dim(pi(Pi)) <= 1 and dim(Pi) = 2, we get dim(Pi cap ell) >= 1, hence ell subset Pi.

Applying this to Pi_2 and Pi_4 (both Lagrangian and contained in V, since Pi_2 = span{ell_{12}, ell_{23}} subset V and Pi_4 = span{ell_{34}, ell_{41}} subset V): ell subset Pi_i for all i = 1, 2, 3, 4. Then every consecutive intersection Pi_i cap Pi_{i+1} contains ell, forcing all edge rays to equal ell. This contradicts the requirement of 4 distinct faces (which need 4 distinct edge rays).

Conclusion: dim(Pi_1 cap Pi_3) = 0. By the identical argument for {Pi_2, Pi_4}: dim(Pi_2 cap Pi_4) = 0. This proves part (b).

STEP 3 (TOPOLOGICAL DISK). The link L = TC_0(K) cap S^3 is the union of 4 arcs: alpha_i = S_i cap S^3 (a great circle arc in Pi_i cap S^3 from ell_{i-1,i} to ell_{i,i+1}), for i = 1, ..., 4. These arcs share endpoints cyclically: alpha_i ends where alpha_{i+1} begins.

Non-adjacent arcs are disjoint: alpha_1 lies in Pi_1 cap S^3 and alpha_3 lies in Pi_3 cap S^3. Since Pi_1 cap Pi_3 = {0} (Step 2), the great circles Pi_1 cap S^3 and Pi_3 cap S^3 are disjoint (any common point p would satisfy p in Pi_1 cap Pi_3 cap S^3, but Pi_1 cap Pi_3 = {0} and 0 not in S^3). So alpha_1 and alpha_3 are disjoint. Similarly alpha_2 and alpha_4 are disjoint (since Pi_2 cap Pi_4 = {0}).

Adjacent arcs share exactly one endpoint: alpha_i and alpha_{i+1} meet at the unit vector in ell_{i,i+1} and at no other point (they lie on different great circles Pi_i cap S^3 and Pi_{i+1} cap S^3, which intersect in at most 2 antipodal points on S^3; the intersection Pi_i cap Pi_{i+1} = ell_{i,i+1} gives exactly 2 antipodal points on S^3, and each arc occupies less than a semicircle, so they share exactly one of the two antipodal points).

Therefore L is a simple closed curve: 4 arcs joined end-to-end with no self-intersections. The cone on L (union of rays from 0 through L) is homeomorphic to the cone on S^1, which is D^2. This proves part (c).

STEP 4 (Sp(4,R) NORMAL FORM). We derive the unique normal form.

Sub-step 4a. Since Pi_1 and Pi_3 are transverse Lagrangian planes with R^4 = Pi_1 oplus Pi_3, choose dual bases {a_1, a_2} of Pi_1 and {b_1, b_2} of Pi_3 with omega(a_i, b_j) = delta_{ij}. Setting e_1 = a_1, e_2 = a_2, e_3 = b_1, e_4 = b_2 gives a Darboux basis with Pi_1 = span{e_1, e_2} and Pi_3 = span{e_3, e_4}.

Sub-step 4b. The stabilizer of (Pi_1, Pi_3) in Sp(4,R) acts as GL(2,R) on Pi_1 (via A -> diag(A, (A^T)^{-1})). The edge ell_{12} is a line in Pi_1; use GL(2,R) to send ell_{12} to R*e_1. The residual stabilizer (preserving R*e_1 in Pi_1) consists of A = [[lambda, mu], [0, nu]] (upper triangular), and the transformation e_2 -> mu*e_1 + nu*e_2 can be used to send ell_{41} (a line in Pi_1 distinct from R*e_1) to R*e_2 (by choosing mu and nu appropriately). After this: ell_{12} = R*e_1 and ell_{41} = R*e_2.

Sub-step 4c. Pi_2 contains ell_{12} = R*e_1 and ell_{23} (a line in Pi_3). The Lagrangian condition forces Pi_2 = span{e_1, alpha*e_1 + e_4} for some alpha (derivation: the second basis vector u of Pi_2 must satisfy omega(e_1, u) = 0, so the e_3-component of u vanishes; the e_2-component also vanishes because dim(Pi_2 cap Pi_3) = 1 forces it; see detailed computation below). The remaining parabolic B-freedom (symmetric shear e_3 -> e_3 + B_{11}*e_1, e_4 -> e_4 + B_{12}*e_1) can absorb alpha: after the shear, Pi_2 = span{e_1, e_4}. So ell_{23} = R*e_4.

Detailed computation for Pi_2: Pi_2 = span{e_1, a*e_1 + b*e_2 + c*e_3 + d*e_4}. Lagrangian: omega(e_1, second vector) = c = 0. For ell_{23} = Pi_2 cap Pi_3: vectors s*e_1 + t*(a*e_1 + b*e_2 + d*e_4) in Pi_3 = span{e_3, e_4} require s + ta = 0 and tb = 0. If b != 0 then t = 0, s = 0, contradicting dim = 1. So b = 0. Then Pi_2 = span{e_1, a*e_1 + d*e_4} = span{e_1, e_4} (after absorbing a*e_1 and normalizing d = 1).

Sub-step 4d. Now ell_{34} is a line in Pi_3 = span{e_3, e_4} distinct from ell_{23} = R*e_4, so ell_{34} = R*(e_3 + mu*e_4). And ell_{41} = R*e_2 is in Pi_4. Pi_4 = span{e_2, e_3 + mu*e_4}. Lagrangian: omega(e_2, e_3 + mu*e_4) = omega(e_2, e_3) + mu*omega(e_2, e_4) = 0 + mu = mu. So mu = 0, giving ell_{34} = R*e_3 and Pi_4 = span{e_2, e_3}.

Sub-step 4e. Residual freedom: after all normalizations, the remaining Sp(4,R) elements preserving (Pi_1, Pi_3, R*e_1, R*e_2, Pi_2, R*e_4) are diag(lambda, nu, 1/lambda, 1/nu). These preserve the entire configuration. So the configuration is fully determined:

  Pi_1 = span{e_1, e_2}, Pi_2 = span{e_1, e_4}, Pi_3 = span{e_3, e_4}, Pi_4 = span{e_2, e_3}.
  Edges: R*e_1, R*e_4, R*e_3, R*e_2.

This is the UNIQUE Sp(4,R)-normal form. The moduli space is 0-dimensional (a single orbit).

Remark (C(gamma) family). In the partial normalization fixing Pi_1 = span{e_1, e_2}, ell_{12} = R*e_1, ell_{41} = R*e_2, Pi_2 = span{e_1, e_4} (but NOT normalizing Pi_3 further), the general form is Pi_3 = span{e_4, gamma*e_1 + e_3}, Pi_4 = span{e_2, gamma*e_1 + e_3}, depending on gamma in R. The symplectomorphism g: e_1 -> e_1, e_2 -> e_2, e_3 -> e_3 - gamma*e_1, e_4 -> e_4 sends C(gamma) to C(0). (Proof that g in Sp(4,R): omega(g(e_1), g(e_3)) = omega(e_1, e_3 - gamma*e_1) = 1, omega(g(e_2), g(e_4)) = omega(e_2, e_4) = 1, all other pairings vanish. Also: g(Pi_3(gamma)) = span{e_4, gamma*e_1 + e_3 - gamma*e_1} = span{e_4, e_3} = Pi_3(0), g(Pi_4(gamma)) = span{e_2, gamma*e_1 + e_3 - gamma*e_1} = span{e_2, e_3} = Pi_4(0), and g fixes Pi_1, Pi_2.) Node 1.3 uses the C(gamma) form for constructive flexibility (the generating-function construction works uniformly for all gamma), but the reader should note all such configurations are equivalent.

STEP 5 (LAGRANGIAN GRASSMANNIAN). The identification LG(n, 2n) = U(n)/O(n) is classical. Identify R^{2n} = C^n via z_j = x_j + i*y_j with omega = Im(sum dz_j wedge d bar{z}_j)/2. The unitary group U(n) preserves omega and acts transitively on Lagrangian n-planes (any Lagrangian plane has an orthonormal basis that extends to a unitary basis of C^n). The stabilizer of the standard Lagrangian plane R^n = {y = 0} in U(n) is O(n). Hence LG(n, 2n) = U(n)/O(n).

For n = 2: LG(2,4) = U(2)/O(2). This is a compact manifold of real dimension dim U(2) - dim O(2) = 4 - 1 = 3.

Fundamental group: from the long exact sequence of the fibration O(2) -> U(2) -> U(2)/O(2):
  pi_1(O(2)) -> pi_1(U(2)) -> pi_1(LG(2,4)) -> pi_0(O(2)) -> pi_0(U(2)).

pi_1(O(2)) = Z (generated by a loop in SO(2) = S^1). pi_1(U(2)) = Z (via det: U(2) -> U(1)). pi_0(O(2)) = Z/2. pi_0(U(2)) = 0.

The map pi_1(O(2)) -> pi_1(U(2)): the generator of pi_1(SO(2)) is gamma(t) = diag(e^{it}, 1) for t in [0, 2pi]. Under inclusion SO(2) -> U(2): gamma(t) has det = e^{it}, which represents the generator of pi_1(U(1)) = Z under the det map. Wait: SO(2) embeds into U(2) as 2x2 real rotation matrices [[cos t, -sin t],[sin t, cos t]], which have det = 1 (as complex matrices), so the loop maps to the TRIVIAL element in pi_1(U(2)).

More carefully: the standard embedding O(2) -> U(2) sends a real orthogonal matrix A to the same matrix viewed as a unitary matrix. The generator of pi_1(SO(2)) is A(t) = [[cos t, -sin t],[sin t, cos t]] (t in [0, 2pi]). As a unitary matrix, det(A(t)) = cos^2(t) + sin^2(t) = 1. So the loop has constant determinant 1 and lies in SU(2). Since pi_1(SU(2)) = 0 (SU(2) = S^3), this loop is contractible in U(2). The map pi_1(O(2)) -> pi_1(U(2)) is the zero map.

The exact sequence becomes: 0 -> pi_1(U(2)) -> pi_1(LG(2,4)) -> pi_0(O(2)) -> 0, i.e., 0 -> Z -> pi_1(LG(2,4)) -> Z/2 -> 0. This is an extension of Z/2 by Z. The extension is determined by the connecting homomorphism in the long exact sequence, which identifies pi_1(LG(2,4)) = Z (the nontrivial extension, corresponding to the index-2 subgroup 2Z in Z with quotient Z/2). Concretely, the generator of pi_1(LG(2,4)) maps to the nontrivial element of pi_0(O(2)) (i.e., its square maps to the generator of the Z subgroup). This gives pi_1(LG(2,4)) = Z.

(Reference: McDuff-Salamon, "Introduction to Symplectic Topology," Section 2.1; or Guillemin-Sternberg, "Symplectic Techniques in Physics.")

This proves part (e).

END OF PROOF.

SUMMARY. At each vertex of a polyhedral Lagrangian surface K (topological manifold, 4 faces per vertex):
  (1) The tangent cone consists of 4 Lagrangian sectors in 4 distinct Lagrangian planes.
  (2) Non-consecutive planes are transverse (Type B is impossible).
  (3) The tangent cone is homeomorphic to a disk (link is a simple closed curve in S^3).
  (4) All Type A configurations are Sp(4,R)-equivalent; the unique normal form is Pi_1 = span{e_1,e_2}, Pi_2 = span{e_1,e_4}, Pi_3 = span{e_3,e_4}, Pi_4 = span{e_2,e_3}.
  (5) LG(2,4) = U(2)/O(2), a compact 3-manifold with pi_1 = Z.

### Node 1.3

**Statement:** Local vertex smoothing via cotangent generating functions (Construction). Depends on: Node 1.2 (vertex classification and normal form). Inference type: construction.

STATEMENT (Lemma 1.3). Let v be a vertex of the polyhedral Lagrangian surface K in (R^4, omega_std) where 4 faces meet. For every delta > 0 sufficiently small, there exists a smooth embedded Lagrangian disk L_v in B_delta(v) and a compactly supported Hamiltonian isotopy phi_H^t with supp(H) subset B_delta(v) such that phi_H^1(K cap B_delta(v)) = L_v. The disk L_v agrees with K on each open sector near partial B_delta(v) (i.e., away from the edge rays).

CONVENTIONS. Coordinates (x_1, x_2, y_1, y_2) on R^4, omega = dx_1 ^ dy_1 + dx_2 ^ dy_2. Identify R^4 = T*R^2 with base X = (x_1, x_2), fiber Y = (y_1, y_2). For any smooth F: U -> R, Graph(dF) := {(X, dF(X)) : X in U} is Lagrangian in T*R^2 (pullback of omega under X -> (X,dF(X)) is d(dF) = 0).

PROOF.

STEP 1 (REDUCTION TO gamma = 0). Translate v to 0. By Node 1.2 (validated), every vertex is Type A. The Sp(4,R)-normal form depends on gamma in R. By Node 1.2(e), the symplectomorphism g(e_1)=e_1, g(e_2)=e_2, g(e_3)=e_3-gamma*e_1, g(e_4)=e_4 sends C(gamma) to C(0). (Check: g preserves omega.) So every vertex is equivalent to gamma=0.

It suffices to construct the smoothing for gamma=0. For general v, conjugate by the normalizing Phi_v in Sp(4,R): reduce to gamma=0, smooth, conjugate back. Symplectomorphisms preserve all relevant properties.

Henceforth, gamma=0:
  Pi_1=span{e_1,e_2}, Pi_2=span{e_1,e_4}, Pi_3=span{e_3,e_4}, Pi_4=span{e_2,e_3}.
  Edges: ell_{12}=R*e_1, ell_{23}=R*e_4, ell_{34}=R*e_3, ell_{41}=R*e_2.

STEP 2 (COTANGENT GRAPH COORDINATES).

Transversality. LG(2,4) has dimension n(n+1)/2 = 3 (for n=2). Each Maslov cycle Sigma(Pi_i) is codimension 1 in LG(2,4). Since LG(2,4) is connected (diffeomorphic to U(2)/O(2)) and 3-dimensional, the complement of finitely many codimension-1 subvarieties is nonempty. So Lambda transverse to all 4 planes exists.

Explicit: Lambda = span{e_1+e_3, e_2+e_4}. Lagrangian check: omega(e_1+e_3, e_2+e_4)=0. Transverse to all Pi_i (determinants: 1, 1, 1, -1).

Darboux basis: f_1=(e_1-e_3)/2, f_2=(e_2-e_4)/2, f_3=e_1+e_3, f_4=e_2+e_4. Verify: omega(f_1,f_3)=1, omega(f_2,f_4)=1, all others zero. Symplectomorphism Phi: (f_1,f_2,f_3,f_4) -> (e_1,e_2,e_3,e_4), giving new coordinates (X_1,X_2,Y_1,Y_2).

Plane images as graphs Y = A_i X:
  A_1 = (1/2)Id, A_2 = diag(1/2,-1/2), A_3 = -(1/2)Id, A_4 = diag(-1/2,1/2).

(From: e_1 = f_1+(1/2)f_3 -> (X,Y)=((1,0),(1/2,0)); e_2 = f_2+(1/2)f_4 -> ((0,1),(0,1/2)); e_3 = -f_1+(1/2)f_3 -> ((-1,0),(1/2,0)); e_4 = -f_2+(1/2)f_4 -> ((0,-1),(0,1/2)). All A_i symmetric (diagonal), confirming Lagrangian.)

Edge projections = semi-axes. Sectors = open quadrants: Omega_1={X_1>0,X_2>0}, Omega_2={X_1>0,X_2<0}, Omega_3={X_1<0,X_2<0}, Omega_4={X_1<0,X_2>0}.

STEP 3 (PL GENERATING FUNCTION). F_i(X) = (1/2)X^T A_i X on each sector:
  F_1=(X_1^2+X_2^2)/4, F_2=(X_1^2-X_2^2)/4, F_3=-(X_1^2+X_2^2)/4, F_4=(-X_1^2+X_2^2)/4.

F_PL := F_i on closure(Omega_i) is continuous (edges match: F_1(X_1,0)=X_1^2/4=F_2(X_1,0), etc.), piecewise smooth, with creases along semi-axes. Graph(dF_PL) over sector interiors = K.

STEP 4 (TWO-ZONE SMOOTH GENERATING FUNCTION).

WHY ANGULAR PARTITION FAILS AT ORIGIN. The old approach F_old(X) = sum_i rho_i(theta)*F_i(X) equals r^2*g(theta). Smoothness at 0 requires g to have Fourier modes only |n|<=2, n even. Generic rho_i introduces all even modes. The "Hessian" sum_i rho_i(theta)*A_i depends on theta — not a well-defined bilinear form. F_old is NOT C^2 at 0.

TWO-ZONE CONSTRUCTION. Let chi: R->[0,1] be smooth, chi(t)=0 for t<=1, chi(t)=1 for t>=2 (flat at both transitions: chi is identically 0 on (-inf,1] and identically 1 on [2,inf), so all one-sided derivatives vanish at t=1 and t=2).

Let {rho_i}_{i=1}^4 be a smooth partition of unity on S^1 subordinate to slightly enlarged quadrants: rho_i=1 deep inside Omega_i, transitions of angular width alpha>0 (arbitrarily small) near sector boundaries.

For any epsilon > 0, define:
  F_angular(X) = sum_{i=1}^4 rho_i(theta(X)) * (1/2)X^T A_i X,    (C^inf on R^2\{0})
  F^{(epsilon)}(X) = chi(|X|/epsilon) * F_angular(X).

SMOOTHNESS OF F^{(epsilon)} ON R^2:

(i) |X|<epsilon: chi=0, F^{(epsilon)}=0. C^inf trivially.

(ii) epsilon<=|X|<=2*epsilon: chi(|X|/epsilon) and F_angular both C^inf (|X|>0). Product C^inf.

(iii) |X|>2*epsilon: chi=1, F^{(epsilon)}=F_angular. C^inf.

(iv) At |X|=epsilon: chi = 0 on (-inf,1], so chi^{(k)}(1)=0 for all k>=0 (including k=0). F_angular = O(|X|^2) and all derivatives bounded near |X|=epsilon. By Leibniz rule, D^alpha(chi*F_angular) -> 0 as |X|->epsilon^+, matching the zero function. C^inf across this boundary.

(v) At the ORIGIN: F^{(epsilon)} = 0 on {|X|<epsilon}. All D^k F^{(epsilon)}(0) = 0. The Hessian D^2F^{(epsilon)}(0) = 0 is a well-defined (zero) bilinear form, direction-independent. This is the key fix.

Set epsilon_0 = delta/8. The smooth generating function is F_smooth := F^{(epsilon_0)}.

STEP 5 (THE SMOOTH LAGRANGIAN L_v). Define:

  L_v = Graph(dF_smooth) restricted to B_delta(0) = {(X, dF_smooth(X)) : X in B_delta(0)}.

Properties:
  (a) L_v is Lagrangian (graph of exact 1-form dF_smooth).
  (b) L_v is C^inf (F_smooth is C^inf on B_delta(0) by Step 4).
  (c) In each sector interior Omega_i, for |X|>2*epsilon_0=delta/4: F_smooth = F_angular, and rho_i=1 deep inside Omega_i, so F_smooth = F_i = F_PL and dF_smooth = A_i X = dF_PL. Hence L_v = K on the open sectors away from sector boundaries and from the origin.
  (d) Near partial B_delta(0) in each sector interior: L_v = K. The surfaces agree on these open arcs.
  (e) Near the 4 edge-boundary intersection points on partial B_delta(0): L_v smoothly interpolates across the crease, deviating from K by O(alpha) (the angular transition width). This angular width alpha>0 can be chosen arbitrarily small.

STEP 6 (SMOOTHNESS VERIFICATION — SUMMARY). F_smooth = F^{(delta/8)} is C^inf on B_delta(0):
  Zone 1 ({|X|<delta/8}): F_smooth=0. All derivatives zero. D^2F_smooth(0)=0 well-defined.
  Zone 2 ({delta/8<=|X|<=delta/4}): chi(8|X|/delta)*F_angular. C^inf (product of C^inf functions, |X|>0).
  Zone 3 ({delta/4<|X|<delta}): F_angular. C^inf (|X|>0).
  Boundaries: C^inf matching by flatness of chi (Step 4(iv)).

No use of F_PL in the definition of F_smooth, so creases of F_PL never enter. Smoothness is unconditional.

STEP 7 (EMBEDDEDNESS). L_v = Graph(dF_smooth) over B_delta(0) is embedded: the graph map iota(X) = (X, dF_smooth(X)) is an injective immersion (projection to X-plane is a left inverse; differential [Id; D^2F_smooth] has rank 2). Any graph of a smooth map over an open domain is automatically an embedded submanifold.

STEP 8 (HAMILTONIAN ISOTOPY).

We construct a compactly supported Hamiltonian isotopy from K to L_v.

KEY IDEA: Vary the inner zone radius epsilon from 0 (giving F_angular = F_PL on sectors, i.e., K) to epsilon_0 = delta/8 (giving F_smooth, i.e., L_v). Use a flat reparametrization so the Hamiltonian extends smoothly.

Let tau: [0,1]->[0,1] be smooth with tau(0)=0, tau(1)=1, and tau flat at 0: tau^{(k)}(0)=0 for all k>=1 (e.g., tau(t)=exp(-1/t)/[exp(-1/t)+exp(-1/(1-t))]). Define:

  epsilon(t) = (delta/8)*tau(t),   t in [0,1].

So epsilon(0)=0 and epsilon(1)=delta/8. For t in (0,1], define the smooth generating function:

  G_t(X) = F^{(epsilon(t))}(X) = chi(|X|/epsilon(t)) * F_angular(X).

Each G_t is C^inf on R^2 (by Step 4 with epsilon=epsilon(t)). The family of smooth Lagrangians:

  L_t = Graph(dG_t),   t in (0,1].

At t=1: G_1 = F^{(delta/8)} = F_smooth, so L_1 = L_v (the target smooth surface).

As t->0+: epsilon(t)->0, so chi(|X|/epsilon(t))->1 for all X!=0 (since |X|/epsilon(t)->inf). Thus G_t(X)->F_angular(X) pointwise on R^2\{0}. On sector interiors, F_angular = F_PL, so L_t -> K on sector interiors.

The time-dependent Hamiltonian generating {L_t} is:

  H_t(X,Y) = (d/dt)G_t(X).

Computing: let s=|X|/epsilon(t). Then G_t = chi(s)*F_angular(X), so

  H_t = (d/dt)chi(s) * F_angular(X) = chi'(s)*(-|X|*epsilon'(t)/epsilon(t)^2)*F_angular(X).

Since chi'(s) is supported on s in [1,2], H_t is supported on {epsilon(t) <= |X| <= 2*epsilon(t)}, an annulus shrinking to {0} as t->0.

SMOOTH EXTENSION TO t=0: epsilon'(t) = (delta/8)*tau'(t). Since tau is flat at 0, tau'(t) (and all higher derivatives) vanish faster than any power of t as t->0. The Hamiltonian H_t involves epsilon'(t)/epsilon(t)^2 = tau'(t)/(delta*tau(t)^2/8). For our choice of tau, this ratio and all its t-derivatives extend smoothly to t=0 with value 0 (tau'(t)/tau(t)^2 -> 0 as t->0 for the standard smooth-step function, since tau(t) ~ exp(-1/t) and tau'(t) ~ (1/t^2)*exp(-1/t), giving tau'/tau^2 ~ exp(1/t)/t^2 ... ).

CORRECTION: the ratio tau'/tau^2 does NOT go to zero for the standard bump function. We need a different argument.

ALTERNATIVE SMOOTH EXTENSION: Instead of analyzing the ratio, we observe directly:

For any fixed X != 0, consider G_t(X) = chi(|X|/epsilon(t)) * F_angular(X) as a function of t. For t large enough that epsilon(t) < |X|/2, we have |X|/epsilon(t) > 2, so chi = 1 and G_t(X) = F_angular(X) (constant in t). So (d/dt)G_t(X) = 0 for t sufficiently large relative to X. And for t small enough that epsilon(t) > |X|, we have chi = 0 and G_t(X) = 0 (constant). So H_t(X,Y) = 0 for t sufficiently small at fixed X != 0.

At X = 0: G_t(0) = 0 for all t (chi(0)*F_angular(0)=0). So H_t(0,Y) = 0 for all t.

Thus, for each (X,Y), H_t(X,Y) is nonzero only for t in a bounded interval (depending on X), and H_t = 0 near t=0 for each fixed X. This means H_t extends smoothly to t=0 with H_0 = 0 (and all time-derivatives zero at t=0).

More precisely: H_t(X,Y) != 0 only when epsilon(t) <= |X| <= 2*epsilon(t), i.e., |X|/(2) <= epsilon(t) <= |X|. Since epsilon(t) is continuous with epsilon(0)=0 and epsilon increasing, for fixed X with |X|>0, the condition epsilon(t) >= |X|/2 requires t >= epsilon^{-1}(|X|/2) > 0. So H_t(X,Y) = 0 for all t < epsilon^{-1}(|X|/2). In particular, H_t vanishes in a neighborhood of t=0 for each X!=0, and H_t(0,Y)=0 for all t. Therefore H extends smoothly to [0,1] x R^4 with H_0 = 0.

The flow phi_H^t is a smooth symplectomorphism for each t in [0,1]. At t=0: phi_H^0 = id. At t=1: phi_H^1 maps L_0 = lim_{t->0+} L_t = K (on sector interiors) to L_1 = L_v.

REVERSING: Define tilde{H}_t = -H_{1-t}. Then phi_{tilde{H}}^1 maps L_v to K. Equivalently, (phi_{tilde{H}}^1)^{-1} = phi_H^1 maps K to L_v.

COMPACT SUPPORT: supp(H_t) subset {|X| <= 2*epsilon(t)} x R^2 subset {|X| <= delta/4} x R^2 subset B_delta(0) for all t.

REMARK: The flow phi_H^t is a smooth symplectomorphism that gradually "opens up" the inner flat zone from radius 0 (surface K, no inner zone) to radius delta/8 (surface L_v, with smooth inner zone). For each t > 0, L_t is smooth. At t = 0, L_0 = K is PL. The isotopy is continuous at t = 0 (uniform convergence of dG_t to dF_PL on compact subsets of sector interiors) and smooth for t > 0.

STEP 9 (BOUNDARY CONDITIONS FOR GLOBAL ASSEMBLY).

(a) SECTOR AGREEMENT: In each open sector Omega_i near partial B_delta(0): rho_i = 1, so F_smooth = F_angular = F_i = F_PL. Thus L_v = K on these open regions. The surfaces agree on the boundary face-by-face.

(b) EDGE-BOUNDARY BEHAVIOR: Near the 4 points where edge rays meet partial B_delta(0), L_v smoothly interpolates across the crease (angular transition of width alpha, arbitrarily small). These narrow strips lie within the tubular neighborhood of the edges, where Node 1.4 operates.

(c) COMPACT SUPPORT: supp(H) subset B_delta(v).

(d) TOPOLOGY: L_v is an embedded disk (graph over B_delta(0)), with boundary curve on partial B_delta(v) that agrees with K's link on the 4 open arcs.

(e) PARAMETER FREEDOM: delta > 0 arbitrary (sufficiently small). Angular width alpha > 0 arbitrary. These coordinate with Node 1.4 (edge smoothing) and Node 1.5 (global assembly).

WORKED EXAMPLE (gamma=0, delta=1, epsilon=1/8).

Matrices: A_1=(1/2)Id, A_2=diag(1/2,-1/2), A_3=-(1/2)Id, A_4=diag(-1/2,1/2).
PL: F_1=(X_1^2+X_2^2)/4 on Omega_1, etc.
Angular: F_angular = (X_1^2/4)(rho_1+rho_2-rho_3-rho_4) + (X_2^2/4)(rho_1-rho_2-rho_3+rho_4). In quadrant interiors: recovers each F_i.
Smooth: F_smooth = chi(8|X|)*F_angular. For |X|<1/8: F_smooth=0. For |X|>1/4: F_smooth=F_angular.
At origin: F_smooth(0)=0, D^2F_smooth(0)=0 (well-defined zero bilinear form).
L_v = Graph(dF_smooth): smooth embedded Lagrangian disk.
Hamiltonian: shrinking inner-zone isotopy, compactly supported in B_1(0).

END OF PROOF.

### Node 1.4

**Statement:** Edge smoothing via explicit Lagrangian profile replacement (Construction). Depends on: Node 1.2 (vertex classification and edge structure). Inference type: construction.

STATEMENT. Let e be an edge of the polyhedral Lagrangian surface K in (R^4, omega_std) where exactly 2 Lagrangian faces F_1, F_2 meet. There exists a smooth embedded Lagrangian surface L_e, defined in a tubular neighborhood of e (away from vertex balls), that replaces the dihedral singularity along e and agrees with the original polyhedral faces outside the smoothing region. The construction is explicit and the Lagrangian condition holds exactly (no correction needed). Endpoint compatibility with vertex smoothings from Node 1.3 is deferred to Node 1.5 (global assembly).

CONVENTIONS. Coordinates (x_1, x_2, y_1, y_2) on R^4 with omega = dx_1 ^ dy_1 + dx_2 ^ dy_2, following Node 1.2.

PROOF.

STEP 1 (EDGE CONFIGURATION). Two Lagrangian faces F_1, F_2 share an edge e. Each face lies in a Lagrangian plane (Pi_1, Pi_2 respectively), and Pi_1 cap Pi_2 contains the edge direction v (a nonzero vector). The two planes are distinct (different faces) but share exactly a line along e (from Node 1.2, Step 1: consecutive planes share a 1-dimensional intersection).

STEP 2 (LINEAR SYMPLECTIC COORDINATES — FROM FIRST PRINCIPLES). We construct a Darboux basis adapted to the edge by LINEAR ALGEBRA in Sp(4,R). This is NOT a neighborhood theorem and does NOT invoke any PL Darboux theorem (which does not exist; cf. Jauberteau-Rollin 2024). The construction uses only the global linear symplectic structure of (R^4, omega_std).

Sub-step 2a. Set e_1 = v/|v| (unit vector along the edge). Since Pi_1 is Lagrangian and contains e_1, we have omega(e_1, w) = 0 for all w in Pi_1. Similarly omega(e_1, w) = 0 for all w in Pi_2. Write Pi_1 = span{e_1, w_1} and Pi_2 = span{e_1, w_2} where w_1, w_2 are linearly independent vectors not parallel to e_1.

Sub-step 2b. Since Pi_1 is Lagrangian: omega(e_1, w_1) = 0. Since Pi_2 is Lagrangian: omega(e_1, w_2) = 0. The symplectic complement of e_1 is the 3-dimensional space {e_1}^omega = ker(omega(e_1, -)). Both w_1 and w_2 lie in {e_1}^omega. Choose f_1 in R^4 with omega(e_1, f_1) = 1. Then f_1 is NOT in {e_1}^omega, and R^4 = {e_1}^omega oplus R*f_1.

Sub-step 2c. Project: the quotient V = {e_1}^omega / R*e_1 is a 2-dimensional symplectic vector space with the induced form omega_bar. The images bar{w}_1, bar{w}_2 of w_1, w_2 in V are nonzero (since Pi_1 != Pi_2, so w_1 and w_2 are not proportional modulo e_1 generically; even if they happen to be parallel mod e_1, we can choose w_1, w_2 to avoid this). Choose e_2 in {e_1}^omega representing bar{w}_1 (i.e., the component of w_1 orthogonal to e_1 in {e_1}^omega). Then omega(e_1, e_2) = 0 and omega(e_2, f_1) is some scalar; adjust f_1 to ensure omega(e_2, f_1) = 0 (replace f_1 by f_1 - omega(e_2, f_1)*e_1/omega(e_1, f_1) if needed — but simpler: choose f_2 with omega(e_2, f_2) = 1 and omega(e_1, f_2) = 0, omega(f_1, f_2) = 0).

Sub-step 2d. Explicitly: choose any f_1 with omega(e_1, f_1) = 1. Set e_2 = w_1 - omega(w_1, f_1)*e_1 (ensuring omega(e_2, f_1) = 0 while keeping e_2 in span of {e_1}^omega). Then choose f_2 with omega(e_2, f_2) = 1, omega(e_1, f_2) = 0, omega(f_1, f_2) = 0. The basis (e_1, e_2, f_1, f_2) is a Darboux basis: omega(e_i, f_j) = delta_{ij}, omega(e_i, e_j) = 0, omega(f_i, f_j) = 0.

Sub-step 2e. In the Darboux coordinates (x_1, x_2, y_1, y_2) defined by this basis:
- The edge e lies along the x_1-axis.
- Pi_1 = span{e_1, e_2} = {y_1 = 0, y_2 = 0}. (By construction, e_2 was chosen from w_1.)
- Pi_2 = span{e_1, w_2}. Since omega(e_1, w_2) = 0 (Lagrangian condition), the f_1-component of w_2 vanishes: w_2 has no y_1-component. So Pi_2 lies in {y_1 = 0}. The projection of w_2 to the (x_2, y_2)-plane is a nonzero vector (since Pi_2 != Pi_1 and the edge direction e_1 is common).

CONCLUSION OF STEP 2. In adapted coordinates:
- The edge is the x_1-axis.
- BOTH faces lie in the hyperplane {y_1 = 0}. This is because both face planes contain e_1, and the Lagrangian condition omega(e_1, -) = 0 forces dy_1 = 0 on both faces (since the f_1 = y_1 direction is the omega-dual of e_1).
- Face F_1 has transverse profile along the ray r_1 = (1, 0) in the (x_2, y_2)-plane (we normalized e_2 along w_1).
- Face F_2 has transverse profile along a ray r_2 = (cos(theta), sin(theta)) in the (x_2, y_2)-plane, where theta != 0 (mod pi) is the angle between the two face profiles.

The two faces, near the edge, look like:
  F_1: {(x_1, t, 0, 0) : t >= 0} (the half-plane x_2 >= 0 in {y_1 = y_2 = 0}),
  F_2: {(x_1, t*cos(theta), 0, t*sin(theta)) : t >= 0} (the half-plane along r_2 in {y_1 = 0}).

Their union is a "V-shaped" surface: a product of the x_1-line with a V-shape (two rays from the origin) in the (x_2, y_2)-plane, embedded in the 3-dimensional hyperplane {y_1 = 0}.

STEP 3 (THE EXPLICIT CONSTRUCTION). Replace the V-shaped profile (two rays r_1, r_2 in the (x_2, y_2)-plane) by a smooth curve gamma connecting r_1 to r_2. Precisely:

Let gamma: R -> R^2 be a smooth embedded curve in the (x_2, y_2)-plane such that:
- gamma(t) = t * r_1 for t >= T (agrees with ray r_1 for large t),
- gamma(t) = |t| * r_2 for t <= -T (agrees with ray r_2 for large |t|),
- gamma is a simple (non-self-intersecting) curve for all t,
- gamma(t) != 0 for all t (the curve avoids the origin — this is the "rounding").

Write gamma(t) = (gamma_1(t), gamma_2(t)). Define the smoothed surface:

  L_e = {(x_1, gamma_1(s), 0, gamma_2(s)) : x_1 in R, s in R}.

STEP 4 (LAGRANGIAN VERIFICATION). Compute the pullback of omega to L_e. The surface is parametrized by (x_1, s), with embedding:

  iota(x_1, s) = (x_1, gamma_1(s), 0, gamma_2(s)).

The differentials are:
  d(x_1) = dx_1,  d(y_1) = d(0) = 0,
  d(x_2) = gamma_1'(s) ds,  d(y_2) = gamma_2'(s) ds.

Pullback:
  iota*(omega) = iota*(dx_1 ^ dy_1) + iota*(dx_2 ^ dy_2)
               = dx_1 ^ 0 + gamma_1'(s) ds ^ gamma_2'(s) ds
               = 0 + gamma_1'(s) gamma_2'(s) (ds ^ ds)
               = 0.

The pullback is IDENTICALLY ZERO for ANY smooth curve gamma. The smoothed surface L_e is exactly Lagrangian. No Moser correction, no symplectic error, no perturbation argument needed. The Lagrangian condition is automatic because: (i) y_1 = 0 everywhere kills dx_1 ^ dy_1, and (ii) x_2 and y_2 both depend on the single parameter s, so dx_2 ^ dy_2 = (function) * ds ^ ds = 0.

STEP 5 (EMBEDDEDNESS). The map iota(x_1, s) = (x_1, gamma_1(s), 0, gamma_2(s)) is an embedding if and only if:
(a) gamma is an embedding (injective immersion): gamma(s_1) = gamma(s_2) implies s_1 = s_2, and gamma'(s) != 0 for all s.
(b) This is automatic: choose gamma to be a simple smooth curve. For example, take gamma to be a convex curve in the (x_2, y_2)-plane that rounds the corner of the V at the origin. Since the two rays r_1 and r_2 span distinct directions, a smooth convex interpolation exists and is automatically injective.

The map iota is then injective (x_1 is recovered from the first coordinate, s from the remaining coordinates via gamma), and its differential has rank 2 everywhere (partial_{x_1} iota = (1,0,0,0) and partial_s iota = (0, gamma_1'(s), 0, gamma_2'(s)) are linearly independent since gamma' != 0).

Therefore L_e is an embedded smooth Lagrangian surface.

STEP 6 (COMPACT SUPPORT ALONG THE EDGE AND TRANSITION ZONES). The construction of Step 3 uses a CONSTANT profile gamma, independent of x_1. In this form, L_e replaces the V-shape everywhere along the edge. For the purposes of Node 1.4, this is the core local construction.

For the global assembly (Node 1.5, Phase B), the profile must be made x_1-dependent in order to:
(a) match the vertex smoothing from Node 1.3 at edge endpoints (where edges meet vertex balls B_delta(v)),
(b) transition from the smoothed profile back to the original PL profile if needed.

When the profile gamma depends on x_1, the Lagrangian condition is no longer automatically satisfied: the pullback acquires a cross-term from the x_1-dependence of gamma. Specifically, if iota(x_1, s) = (x_1, gamma_1(x_1, s), 0, gamma_2(x_1, s)), then:

  iota*(omega) = dx_1 ^ d(0) + (partial_{x_1} gamma_1 dx_1 + partial_s gamma_1 ds) ^ (partial_{x_1} gamma_2 dx_1 + partial_s gamma_2 ds)
               = (partial_{x_1} gamma_1 * partial_s gamma_2 - partial_s gamma_1 * partial_{x_1} gamma_2) dx_1 ^ ds.

This vanishes iff partial_{x_1} gamma_1 * partial_s gamma_2 = partial_s gamma_1 * partial_{x_1} gamma_2, which is the condition that the x_1-flow preserves the curve gamma up to reparametrization. Equivalently, the x_1-deformation of gamma must be through REPARAMETRIZATIONS of the curve (not deformations of its image). This is a nontrivial matching condition.

RESOLUTION: The endpoint matching and x_1-dependent transition are properly the concern of Node 1.5 (global assembly). Node 1.4 establishes the LOCAL edge smoothing with constant profile: for any finite segment of the edge bounded away from vertices, the V-shape can be replaced by a smooth Lagrangian with the same boundary. Node 1.5 handles the transition zones where the edge meets vertex balls. This separation is appropriate because:
- The constant-profile construction is the core mathematical content (Lagrangian condition for free).
- The transition matching requires information about the vertex smoothing (Node 1.3 output) that Node 1.4 should not assume.

STEP 7 (HAMILTONIAN ISOTOPY). Define a 1-parameter family of curves gamma_t (t in [0,1]) in the (x_2, y_2)-plane:
- gamma_0 is the V-shape (two rays r_1, r_2 meeting at the origin).
- gamma_1 = gamma (the smooth rounded curve from Step 3).
- Each gamma_t is a simple curve coinciding with r_1 and r_2 outside a compact set.

For each t, the surface L_t = {(x_1, gamma_t(s), 0) : x_1, s in R} is Lagrangian (by the computation of Step 4, which is independent of the choice of curve). So {L_t}_{t in [0,1]} is a Lagrangian isotopy from K (restricted to the edge region) to L_e.

Since we work in R^4 (which is simply connected with H^1(R^4; R) = 0), every Lagrangian isotopy with compact support is Hamiltonian: the flux homomorphism vanishes identically. The isotopy {L_t} has compact support in the (x_2, y_2)-plane (gamma_t agrees with the V-shape outside a compact set), so it is generated by a Hamiltonian H_t that is compactly supported in the transverse directions. The x_1-direction is untouched (the profile is constant in x_1).

More explicitly: the isotopy velocity field X_t is tangent to the (x_2, y_2)-plane (since only the profile varies) and is Hamiltonian because iota_{X_t} omega = -dH_t for a function H_t depending only on (x_2, y_2). The function H_t has compact support in the (x_2, y_2)-plane (the isotopy is stationary outside a compact region where gamma_t differs from the V-shape).

STEP 8 (COMPATIBILITY WITH NODE 1.2 AND NODE 1.3).

The edge smoothing (this node) applies to the portion of each edge OUTSIDE all vertex balls. In the constant-profile construction, L_e already matches K outside the smoothing region, ensuring compatibility. ### Node 1.5

**Statement:** Global assembly of local smoothings into a smooth embedded Lagrangian (Construction). Depends on: Node 1.2 (vertex classification and normal form), Node 1.3 (vertex smoothing via two-zone cotangent generating functions), Node 1.4 (edge smoothing via explicit Lagrangian profile replacement). Inference type: construction.

STATEMENT (Proposition). Let K be a finite polyhedral Lagrangian surface in (R^4, omega_std) with exactly 4 faces at every vertex. There exists a smooth embedded Lagrangian surface L and a compactly supported Hamiltonian H: R^4 -> R (with supp(H) contained in an arbitrarily small neighborhood of the 1-skeleton of K) such that the time-1 flow phi_H^1 satisfies phi_H^1(K) = L. Moreover, L agrees with K on the complement of the 1-skeleton neighborhood.

SCOPE. K is a FINITE polyhedral complex (finitely many cells), hence compact. K has finitely many vertices v_1, ..., v_N and finitely many edges e_1, ..., e_M. (A finite polyhedral complex has finitely many cells, each compact, so their union is compact. The non-compact case does not arise under the problem hypotheses.) A brief remark on the locally-finite extension appears in Step 10.

OVERALL STRATEGY. We construct the smooth Lagrangian surface L DIRECTLY (not via Hamiltonian flow applied to K). The construction defines L piecewise on overlapping regions, verifying smoothness at every overlap. A Hamiltonian isotopy from K to L is then constructed separately. This avoids the fundamental difficulty that a Hamiltonian flow applied to a non-smooth surface need not produce a smooth result.

PROOF.

STEP 1 (PARAMETER CHOICES). For each vertex v_i (i = 1, ..., N), choose delta_i > 0 satisfying:
(a) delta_i < (1/6) * min_{j != i} d(v_i, v_j). (The factor 1/6 provides room for extensions while guaranteeing disjointness, see Step 4.)
(b) delta_i is small enough that B_{3*delta_i}(v_i) intersects only edges and faces incident to v_i.
(c) delta_i is small enough for Node 1.3's construction to apply with radius parameter 3*delta_i/2.
Such delta_i exist because K is compact with finitely many vertices.

Choose a uniform small parameter eta with 0 < eta < (1/10)*min_i delta_i. This is the overlap width.

For each edge e_k connecting vertices v_i and v_j, let L_k denote the edge length. Choose epsilon_k > 0 (transverse neighborhood radius) satisfying:
(d) The tubular neighborhoods T_{epsilon_k}(e_k) are pairwise disjoint for non-adjacent edges, and for edges sharing a vertex, their tubular neighborhoods are disjoint outside the vertex ball.
(e) epsilon_k < eta.

STEP 2 (DOMAIN DECOMPOSITION). Partition R^4 near the 1-skeleton into three types of regions:

VERTEX REGIONS: V_i = B_{delta_i + eta}(v_i) for each vertex v_i. The extended vertex balls have pairwise disjoint closures: (delta_i + eta) + (delta_j + eta) < (1/6 + 1/60)*d_min + (1/6 + 1/60)*d_min < (2/5)*d_min < d_min for any pair v_i, v_j (using delta_i < d_min/6 and eta < delta_i/10 < d_min/60).

EDGE REGIONS: For each edge e_k connecting v_i and v_j, define the edge middle E_k = {p on e_k : d(p, v_i) > delta_i and d(p, v_j) > delta_j}, with tubular neighborhood T_k = T_{epsilon_k}(E_k). This is the portion of the edge strictly outside the original (non-extended) vertex balls.

OVERLAP STRIPS: For each edge e_k at vertex v_i, the overlap strip O_{k,i} is the annular segment where the edge middle's tubular neighborhood intersects the extended vertex ball: O_{k,i} = T_k cap V_i. This is a strip of width eta along the edge, at distance delta_i to delta_i + eta from v_i.

FACE REGIONS: Everything else — the open face interiors away from all edges and vertex balls.

STEP 3 (VERTEX SMOOTHING — APPLYING NODE 1.3). For each vertex v_i, apply Node 1.3's two-zone cotangent generating function construction with effective radius delta_i + eta (instead of delta_i).

Node 1.3 (rewritten, two-zone construction) provides: a smooth generating function F_smooth^{(i)} defined on B_{delta_i + eta}(v_i) in cotangent coordinates (X_1, X_2) centered at v_i, with:
(i) F_smooth^{(i)}(X) = 0 for |X| < epsilon_0 where epsilon_0 = (delta_i + eta)/8. (Inner zone: F=0, all derivatives zero, Hessian is the well-defined zero matrix.)
(ii) F_smooth^{(i)}(X) = chi(|X|/epsilon_0) * F_angular(X) for all X, where chi is flat at t=1 and equals 1 for t >= 2, and F_angular = sum_j rho_j(theta) * (1/2) X^T A_j X with {rho_j} a smooth angular partition of unity.
(iii) For |X| > 2*epsilon_0 = (delta_i+eta)/4, F_smooth^{(i)} = F_angular, smooth since |X| > 0.
(iv) On each sector interior Omega_j (the open quadrant corresponding to face F_j), deep inside (angular distance > alpha from sector boundaries): rho_j = 1, so F_smooth^{(i)} = F_j = (1/2) X^T A_j X = F_PL.
(v) Near sector boundaries (within angular width alpha of edge rays): F_smooth^{(i)} smoothly interpolates between the adjacent face generating functions via the angular partition of unity.

Define the vertex-smoothed surface piece:
L^{vert}_i = Graph(dF_smooth^{(i)}) = {(X, dF_smooth^{(i)}(X)) : X in B_{delta_i + eta}(v_i)}.

This is a smooth embedded Lagrangian disk (Node 1.3 Steps 5-7).

CRITICAL PROPERTY AT THE BOUNDARY (for overlap matching): On the boundary of V_i = B_{delta_i + eta}(v_i), within each sector interior (angular distance > alpha from edge rays), F_smooth^{(i)} = F_PL. The surface L^{vert}_i agrees EXACTLY with K on these arcs.

Near the boundary of V_i, close to an edge ray (within angular width alpha): F_smooth^{(i)} = F_angular (since |X| > (delta_i+eta)/4 >> epsilon_0). The surface L^{vert}_i is a smooth curve in the fiber coordinates, interpolating between the two adjacent face planes. The angular width alpha is a free parameter that we choose sufficiently small (alpha << epsilon_k).

STEP 4 (EDGE SMOOTHING — APPLYING NODE 1.4). For each edge e_k, we construct the edge-smoothed surface piece using Node 1.4's constant-profile construction.

In Node 1.4's edge-adapted Darboux coordinates (x_1, x_2, y_1, y_2):
- The edge e_k lies along the x_1-axis.
- Both faces meeting at e_k lie in the hyperplane {y_1 = 0}.
- Face F_a has transverse profile along ray r_a = (1, 0) in the (x_2, y_2)-plane.
- Face F_b has transverse profile along ray r_b = (cos theta, sin theta) in the (x_2, y_2)-plane.

The PL surface along the edge is {(x_1, t*r, 0) : x_1 in R, t >= 0, r in {r_a, r_b}} — a product of the x_1-line with a V-shape (two rays) in the (x_2, y_2)-plane.

Choose a smooth embedded curve gamma: R -> R^2 in the (x_2, y_2)-plane satisfying:
(A) gamma(s) = s * r_a for s >= T (flat transition: all derivatives match the linear ray at s = T; this is automatic since gamma^{(k)}(T) = 0 for k >= 2 from the linear ray side, and we require gamma^{(k)}(T^-) = 0 for k >= 2 from the rounded side — achievable via a flat/bump-function construction).
(B) gamma(s) = |s| * r_b for s <= -T (similarly flat transition at s = -T).
(C) gamma is a simple curve avoiding the origin: gamma(s) != 0 for all s.
(D) gamma is smooth on all of R.

Such a curve exists by standard smooth approximation theory: the V-shape (two rays meeting at the origin at angle theta) can be rounded by replacing the corner with a smooth arc of arbitrarily small radius, with flat (all derivatives matching) transitions to the rays at s = +-T.

Define the edge-smoothed surface piece:
L^{edge}_k = {(x_1, gamma_1(s), 0, gamma_2(s)) : x_1 in E_k, s in R}
where gamma(s) = (gamma_1(s), gamma_2(s)).

This is smooth and EXACTLY Lagrangian by Node 1.4, Step 4: the pullback omega|_{L^{edge}_k} = 0 identically because dy_1 = 0 (killing dx_1 ^ dy_1) and x_2, y_2 both depend on the single parameter s (killing dx_2 ^ dy_2). The profile is constant (independent of x_1), so no cross-terms arise.

TRANSVERSE COMPACT SUPPORT: For |s| >= T, L^{edge}_k agrees exactly with K (the V-shape). The smoothing modifies K only in the transverse strip |s| < T near the edge. Outside this strip, L^{edge}_k = K (flat face).

STEP 5 (OVERLAP MATCHING — THE KEY STEP). We must verify that L^{vert}_i and L^{edge}_k agree (define the same smooth surface) on the overlap strip O_{k,i}.

COORDINATE RELATIONSHIP: Node 1.2's normal form puts the vertex at the origin with the 4 faces as Graph(A_j) over the 4 quadrants. Node 1.4's edge-adapted coordinates are a Darboux basis adapted to the edge. Both are LINEAR symplectomorphisms of R^4 (no nonlinear coordinate changes). The transition between them is a CONSTANT ELEMENT of Sp(4, R).

Consider the edge e_k = ell_{j,j+1} between face j and face j+1, which in the vertex-centered coordinates is a coordinate semi-axis. In the overlap strip O_{k,i}, a point on the surface at distance r from v_i (with delta_i < r < delta_i + eta) and small angular distance phi from the edge ray has:
- Vertex coordinates: X = r*(cos phi, sin phi) (approximately, in the sector near the edge ray).
- The generating function: F_smooth^{(i)}(X) = F_angular(X) = rho_j(phi) * F_j(X) + rho_{j+1}(phi) * F_{j+1}(X) + (small corrections from distant sectors).

Near the edge ray (phi near 0), only the two adjacent sectors contribute:
F_angular(X) approx rho_j(phi) * (1/2) X^T A_j X + rho_{j+1}(phi) * (1/2) X^T A_{j+1} X.

The surface Graph(dF_angular) near the edge, expressed in the edge-adapted coordinates of Node 1.4, has the following form. In the transverse (x_2, y_2)-plane, at longitudinal position x_1 = r (distance along the edge from vertex), the cross-section of Graph(dF_angular) is a smooth curve that interpolates between the two face rays r_a, r_b. This curve depends on the angular partition of unity rho_j.

KEY CHOICE: We choose the profile curve gamma for the edge smoothing (Step 4) to MATCH the cross-section of Node 1.3's angular interpolation at the inner boundary of the overlap strip. Specifically:

At longitudinal position x_1 corresponding to |X| = delta_i (the inner boundary of the overlap), Node 1.3's construction produces a specific smooth curve gamma_A(s) in the (x_2, y_2)-plane as the cross-section of Graph(dF_angular). We set gamma := gamma_A. Since Node 1.3's construction is invariant along the edge direction (F_angular depends only on the angular coordinate theta and the radial coordinate |X|, and for points near the edge at fixed |X|, the angular interpolation produces the same curve), the cross-section gamma_A is actually INDEPENDENT of x_1 for points near the edge. Therefore gamma_A = gamma is a valid constant profile.

More precisely: in the vertex-centered cotangent coordinates, the edge ray is a semi-axis. For points X = (r, phi) (polar) near this semi-axis, the generating function F_angular = rho_j(phi) * q_j(X) + rho_{j+1}(phi) * q_{j+1}(X). In the radial-times-angular decomposition, the angular interpolation rho_j(phi) depends ONLY on phi (not on r). The quadratic forms q_j(X) = (1/2) X^T A_j X are degree 2 in |X|. Therefore the tangent to Graph(dF_angular) in the transverse direction has a profile that scales with r but whose SHAPE (as a curve in the projectivized transverse plane) is r-independent. Concretely, the cross-section at radius r is r times the cross-section at radius 1. After rescaling to the edge-adapted coordinates, the profile gamma_A(s) at longitudinal position x_1 is the SAME curve for all x_1 > 0 (up to the overall scale set by the distance from the vertex, which is absorbed into the parametrization).

This means that in the overlap strip, the vertex-smoothed surface L^{vert}_i and the edge-smoothed surface L^{edge}_k define the SAME subset of R^4, provided gamma is chosen to match gamma_A.

VERIFICATION: In the overlap strip O_{k,i}, express both surfaces in the edge-adapted coordinates:
- L^{vert}_i = Graph(dF_angular) restricted to the overlap. Near the edge, this is {(x_1, gamma_A(s), 0, ...) : ...} where gamma_A is the angular interpolation profile.
- L^{edge}_k = {(x_1, gamma(s), 0, gamma_2(s)) : ...} with gamma = gamma_A.
These are the same surface. The overlap is smooth because both descriptions are smooth.

SUBTLETY — FIBER COORDINATES: The cotangent graph form Graph(dF_angular) is expressed in cotangent coordinates (X, Y = dF_angular(X)), while the product form uses edge-adapted coordinates. The coordinate transformation between them is a fixed linear symplectomorphism (composition of the vertex-to-standard and standard-to-edge-adapted changes). Under this linear map, Graph(dF_angular) near the edge IS a constant-profile surface in {y_1 = 0}. This is because:
(1) Both faces lie in {y_1 = 0} in the edge-adapted coordinates (Node 1.4, Step 2).
(2) The angular interpolation between the two face planes stays in {y_1 = 0} (since both face planes have y_1 = 0, any convex combination also has y_1 = 0; the angular partition of unity gives a smoothly-weighted convex combination of the transverse profiles).
(3) The resulting curve in the (x_2, y_2)-plane is gamma_A, which is smooth.

Therefore the overlap matching is exact.

STEP 6 (FACE MATCHING). On face interiors (angular distance > alpha from all edge rays, distance > delta_i + eta from all vertices):
- The vertex smoothing gives F_smooth^{(i)} = F_PL (since rho_j = 1 deep inside sector j). So L^{vert}_i = K.
- The edge smoothing gives gamma = ray (since for |s| >= T, gamma agrees with the face ray). So L^{edge}_k = K.
- L = K on face interiors.

The face-vertex boundary: for points on a face at distance delta_i + eta from v_i and angular distance > alpha from edges, L^{vert}_i = K = L (face interior). Smooth.

The face-edge boundary: for points on a face at transverse distance T from an edge, L^{edge}_k = K = L (face interior). Smooth. The flat transition property of gamma at s = +-T (Step 4(A)(B)) ensures all derivatives match: gamma^{(k)}(T) matches the k-th derivative of the linear ray, which is 0 for k >= 2. So the transition from rounded profile to flat face is C^infinity.

STEP 7 (DEFINITION OF L AND GLOBAL SMOOTHNESS). Define L as follows:
- On V_i: L = L^{vert}_i = Graph(dF_smooth^{(i)}).
- On T_k setminus (union_i V_i): L = L^{edge}_k.
- On face interiors: L = K.

This is well-defined because:
(a) On the overlap strips O_{k,i} = T_k cap V_i: L^{vert}_i = L^{edge}_k (Step 5).
(b) On face-vertex boundaries: L^{vert}_i = K (Step 6).
(c) On face-edge boundaries: L^{edge}_k = K (Step 6).

Smoothness check at every point of L:

(A) Points inside V_i, away from T_k: L = Graph(dF_smooth^{(i)}), smooth (Node 1.3 Step 6).

(B) Points inside V_i, near an edge (in the overlap O_{k,i}): L = L^{vert}_i = L^{edge}_k (both smooth, same surface). Smooth.

(C) Points on T_k outside all V_i (edge middle proper): L = L^{edge}_k = {(x_1, gamma(s), 0, gamma_2(s))}, smooth and exactly Lagrangian by Node 1.4.

(D) Points on the boundary of V_i, away from edges: L transitions from Graph(dF_smooth^{(i)}) = Graph(dF_PL) to K = Graph(dF_PL). These are the same surface. Smooth.

(E) Points on the boundary of T_k, away from vertex balls (face-edge boundary): L transitions from L^{edge}_k to K. At the transverse boundary |s| = T, gamma matches the face ray with all derivatives agreeing (flat transition). L = K on both sides. Smooth.

(F) Points on face interiors: L = K. These are flat Lagrangian planes. Smooth.

All points of L are covered. L is a smooth surface.

STEP 8 (LAGRANGIAN VERIFICATION — GENERATING FUNCTION ARGUMENT). L is Lagrangian everywhere:

(A) In vertex regions V_i: L = Graph(dF_smooth^{(i)}). The graph of an exact 1-form dF is Lagrangian: omega = d(p dq) and Graph(dF) is the image of the section q -> (q, dF(q)), pulling back the Liouville form to dF, so pullback of omega = d(dF) = 0.

(B) On edge middles (inside T_k, outside V_i): L = {(x_1, gamma(s), 0, gamma_2(s))}. Pullback of omega = dx_1 ^ dy_1 + dx_2 ^ dy_2 = dx_1 ^ 0 + gamma_1'ds ^ gamma_2'ds = 0 (Node 1.4, Step 4).

(C) On face interiors: L = K is a union of flat Lagrangian planes. Each face is a Lagrangian subspace.

(D) On overlap strips O_{k,i}: by Step 5, L = L^{vert}_i which is Lagrangian by (A). Equivalently, L = L^{edge}_k which is Lagrangian by (B). Both give the same result.

The Lagrangian condition is verified at EVERY point of L using the appropriate local description. No Hamiltonian flow preservation argument is needed — each piece is Lagrangian by direct computation.

STEP 9 (EMBEDDEDNESS). L is embedded (a smooth injective immersion that is a homeomorphism onto its image):

(a) LOCAL EMBEDDEDNESS: Each piece of L is locally embedded.
- Vertex regions: Graph(dF_smooth^{(i)}) is a graph over B_{delta_i+eta}(v_i), hence injective (the projection to the base X-plane is a left inverse). The differential has rank 2 (the identity block in the X-directions).
- Edge regions: {(x_1, gamma(s), 0, gamma_2(s))} is injective if gamma is injective (x_1 is recovered from the first coordinate, s from gamma). Since gamma is a simple curve (Step 4(C)), the map is injective with rank 2.
- Face regions: flat planes, trivially embedded.

(b) GLOBAL INJECTIVITY (no self-intersections): K is an embedded topological submanifold of R^4 (hypothesis). L is obtained from K by a local modification: the modification occurs only in a small neighborhood of the 1-skeleton, and in each region, L is close to K (the generating function correction is small, the curve gamma is close to the V-shape). For sufficiently small delta_i, eta, and epsilon_k, the modified surface L remains embedded. More precisely:

(i) L is a smooth 2-manifold (verified in Step 7).
(ii) L is a proper map into R^4: K is compact (FINITE polyhedral complex), and L is a local modification of K within bounded neighborhoods, so L is also compact, hence its inclusion into R^4 is proper (closed in R^4).
(iii) L is an injective immersion: at each point, the local descriptions in (a) show injectivity and rank 2.
(iv) By the standard result (a proper injective immersion is an embedding), L is an embedded submanifold.

Alternatively: the generating function F_smooth^{(i)} on vertex balls and the product parametrization on edge middles each provide an explicit local parametrization of L. These parametrizations have pairwise compatible domains (either disjoint or agreeing on overlaps by Step 5). The atlas of local parametrizations gives L the structure of a smooth embedded submanifold.

STEP 10 (HAMILTONIAN ISOTOPY). We now construct a compactly supported Hamiltonian isotopy from K to L. The strategy: compose vertex isotopies (from Node 1.3) with edge isotopies (from Node 1.4), using a smooth time reparametrization.

PHASE A ISOTOPY (VERTEX SMOOTHING): Node 1.3 (Step 8) provides, for each vertex v_i, a Hamiltonian H_{v_i}(t, X, Y) compactly supported in V_i that generates a flow phi_t^{H_{v_i}} with phi_1^{H_{v_i}} mapping K cap V_i to L^{vert}_i.

Since the extended vertex balls V_i = B_{delta_i+eta}(v_i) are pairwise disjoint (Step 2), the Hamiltonians H_{v_1}, ..., H_{v_N} have pairwise disjoint supports. The Poisson brackets vanish: {H_{v_i}, H_{v_j}} = 0. The flow of the sum H_A = sum_i H_{v_i} equals the composition in any order: phi_t^{H_A} = phi_t^{H_{v_1}} o ... o phi_t^{H_{v_N}} for all t (BCH formula with all commutators zero; disjoint supports imply vanishing Lie brackets of Hamiltonian vector fields [X_{H_{v_i}}, X_{H_{v_j}}] = X_{\{H_{v_i}, H_{v_j}\}} = 0).

Define K_A = phi_1^{H_A}(K). Properties: (i) inside V_i: K_A = L^{vert}_i (smooth Lagrangian disk); (ii) outside all V_i: K_A = K (unchanged PL surface).

PHASE B ISOTOPY (EDGE SMOOTHING): For each edge e_k, Node 1.4 (Step 7) provides a family of curves gamma_t (t in [0,1]) from gamma_0 (V-shape) to gamma_1 (smooth curve), generating a Lagrangian isotopy of the constant-profile surface. The isotopy is Hamiltonian with Hamiltonian H_{e_k}^{transverse}(t, x_2, y_2) compactly supported in the (x_2, y_2)-plane.

For Phase B to act only on the edge middle and not interfere inside vertex balls, we multiply by a longitudinal cutoff. However, multiplying the Hamiltonian by a cutoff function chi_long(x_1) does NOT break the Lagrangian condition of the TARGET surface (the target is defined directly in Step 4, independently of the Hamiltonian). The Hamiltonian chi_long * H_{e_k}^{transverse} is a smooth function on R^4, and its flow is a smooth symplectomorphism. The key point: we apply this flow to K_A, which inside the overlap O_{k,i} is ALREADY the smooth surface L^{vert}_i = L^{edge}_k. At points in the overlap where chi_long is ramping from 0 to 1:

- chi_long = 0: the flow is the identity, K_A is unchanged = L^{vert}_i = L (already the target).
- chi_long = 1: the flow fully resolves the V-shape, producing L^{edge}_k = L (the target).
- 0 < chi_long < 1: the flow PARTIALLY resolves the V-shape. But in the overlap, K_A is already smooth (from Phase A), and chi_long * H_{e_k} generates a smooth Hamiltonian flow. So the partial flow applied to a smooth surface gives a smooth surface. However, this smooth surface may NOT equal L in the overlap.

CORRECTION: To avoid this subtlety, we choose chi_long so that it transitions from 0 to 1 OUTSIDE the overlap strip, entirely within the edge middle proper (where K_A = K is PL). Specifically, chi_long(x_1) = 0 for x_1 <= delta_i + eta (inside vertex ball) and chi_long(x_1) = 1 for x_1 >= delta_i + eta + eta' (a small distance into the edge middle), where eta' < sigma/2 is chosen so the transition occurs in the PL region.

But wait — if chi_long = 0 in the overlap, then Phase B does not act in the overlap. Inside the overlap, K_A = L^{vert}_i, which IS already L (by Step 5, the vertex smoothing matches the edge smoothing in the overlap). So Phase B does not NEED to act in the overlap. Outside the overlap and inside the edge middle, K_A = K (PL), and chi_long = 1, so Phase B fully resolves the V-shape, producing L^{edge}_k = L.

In the transition strip (width eta' near the vertex ball boundary, inside the edge middle): K_A = K (PL), and 0 < chi_long < 1. The flow chi_long * H_{e_k} partially resolves the V-shape, producing a surface that is neither fully PL nor fully smooth. This is a PROBLEM.

RESOLUTION: We make chi_long transition inside the OVERLAP, where K_A is already smooth. Set chi_long(x_1) = 0 for x_1 <= delta_i + eta/3 and chi_long(x_1) = 1 for x_1 >= delta_i + 2*eta/3. The transition occurs in [delta_i + eta/3, delta_i + 2*eta/3], which is inside V_i (since delta_i + 2*eta/3 < delta_i + eta). In this region, K_A = L^{vert}_i (smooth from Phase A). The flow of chi_long * H_{e_k} applied to K_A:
- Where chi_long = 0: surface = K_A = L^{vert}_i = L. Correct.
- Where chi_long = 1 and K_A = K (PL, on edge middle): flow fully resolves V-shape, surface = L^{edge}_k = L. Correct.

### Node 1.6

**Statement:** Hamiltonian isotopy verification (Lemma). Depends on: Node 1.3 (vertex smoothing via cotangent generating functions), Node 1.4 (edge smoothing via Lagrangian rounding), Node 1.5 (global assembly). Inference type: lemma.

STATEMENT. The Lagrangian isotopy K_t constructed in Nodes 1.3–1.5 is Hamiltonian. That is, there exists a (possibly time-dependent) Hamiltonian H_t: R^4 -> R such that the flow phi_t^H satisfies phi_t^H(K) = K_t for t in [0,1], with K_0 = K and K_1 smooth Lagrangian.

PROOF.

The argument proceeds in four steps: (1) each local smoothing is Hamiltonian by construction, (2) the global Hamiltonian is a well-defined sum with smooth time reparametrization for concatenation, (3) the Hamiltonian flow is complete (including the non-compact case), (4) the Lagrangian flux vanishes.

STEP 1 (LOCAL SMOOTHINGS ARE HAMILTONIAN BY CONSTRUCTION).

Step 1a (Vertex smoothing — Node 1.3, two-zone construction). At each vertex v, Node 1.3 (rewritten, Session 5) constructs a smooth generating function via a TWO-ZONE construction:

  F_smooth(X) = F^{(epsilon)}(X) = chi(|X|/epsilon) * F_angular(X),

where chi: R -> [0,1] is smooth with chi(t) = 0 for t <= 1, chi(t) = 1 for t >= 2 (flat at both transitions), F_angular(X) = sum_i rho_i(theta(X)) * (1/2) X^T A_i X is smooth on R^2 \ {0}, and epsilon = delta/8.

SMOOTHNESS OF F_smooth: (i) For |X| < epsilon: F_smooth = 0, trivially C^infinity. All derivatives are zero at the origin, including D^2 F_smooth(0) = 0 (a well-defined zero bilinear form). (ii) For epsilon <= |X| <= 2*epsilon: product of two C^infinity functions (chi and F_angular, both smooth since |X| > 0). (iii) For |X| > 2*epsilon: F_smooth = F_angular, smooth. (iv) At the boundary |X| = epsilon: chi and all its derivatives vanish, so by Leibniz rule the product matches the zero function from the inner zone to all orders. The old challenges about angular partition of unity smoothness at the origin (ch-718a9a57ae5, ch-7313ffe7860, ch-34ff33544ce) are ALL RESOLVED by this construction: the inner zone avoids the origin entirely.

The Hamiltonian isotopy is generated by the shrinking-inner-zone family (Node 1.3, Step 8). Define G_t(X) = F^{(epsilon(t))}(X) = chi(|X|/epsilon(t)) * F_angular(X) where epsilon(t) = (delta/8)*tau(t) and tau is a smooth step function flat at t=0. The time-dependent Hamiltonian is H_v(X,Y) = (d/dt)G_t(X), which is:
- supported on the shrinking annulus {epsilon(t) <= |X| <= 2*epsilon(t)},
- smooth (for each fixed X != 0, H_v is nonzero only for t in a bounded interval, and vanishes near t = 0 since epsilon(t) < |X|/2 for t small; at X = 0, H_v = 0 for all t),
- compactly supported in B_delta(v) (since 2*epsilon(t) <= delta/4 < delta for all t).

Step 1b (Edge smoothing — Node 1.4, constant-profile construction). Along each edge e, Node 1.4 replaces the V-shaped transverse profile with a smooth curve gamma(s) in the (x_2, y_2)-plane. In the edge-adapted Darboux coordinates (Node 1.4, Step 2), BOTH faces lie in the hyperplane {y_1 = 0}. The smoothed surface L_e = {(x_1, gamma_1(s), 0, gamma_2(s))} is exactly Lagrangian for ANY smooth curve gamma, because:

  iota*(omega) = dx_1 ^ d(0) + gamma_1'(s) ds ^ gamma_2'(s) ds = 0.

The Lagrangian condition is AUTOMATIC from the product structure (dy_1 = 0 kills the first term; both x_2 and y_2 depend on the single parameter s, making ds ^ ds = 0). No correction of any kind is needed — in particular, NO Weinstein correction. Node 1.4 uses a CONSTANT profile along the edge (no x_1-dependent cutoff), and the Lagrangian condition holds EXACTLY.

The smoothing isotopy interpolates: gamma_t for t in [0,1], where gamma_0 is the V-shape and gamma_1 is the smooth curve. Each intermediate surface L_t = {(x_1, gamma_t(s), 0, gamma_t(s))} is Lagrangian. The Hamiltonian H_e generating this isotopy depends only on the transverse coordinates (x_2, y_2) and has compact support in the (x_2, y_2)-plane (the isotopy is stationary outside the region where gamma_t differs from the V-shape).

For FINITE edges: The profile replacement is constant along x_1 throughout the support of Phase B (which is bounded away from vertex balls, per Node 1.5 Step 5). The Lagrangian condition is automatic. The Hamiltonian H_e is supported in {d(p, e^o) < epsilon_e} (a tubular neighborhood of the edge exterior, bounded away from vertex balls). Since the edge segment is compact, the support is compact.

For INFINITE edges (non-compact K): H_e depends only on (x_2, y_2) and is not compactly supported in x_1. However, H_e is bounded with bounded derivatives of all orders, and the Hamiltonian vector field X_{H_e} is tangent to the transverse (x_2, y_2)-plane with bounded norm. The flow is complete (see Step 3 Case 2 below).

COMPATIBILITY NOTE: The vertex-to-edge transition is handled by Node 1.5's two-phase sequential strategy. Phase A (vertex smoothing) smooths the surface inside vertex balls. Phase B (edge smoothing) acts on edge exteriors that are BOUNDED AWAY from vertex balls. At the Phase B support boundary near a vertex ball, H_B = 0 (Phase B does nothing), and the surface is already smooth from Phase A. At the other boundary (along edge interiors), the constant profile is applied. The junction is smooth because Phase A has already ensured the surface is smooth past the vertex ball boundary (Node 1.3 smoothes the crease in the annular zone), and Phase B begins acting only where the surface is still PL. See Node 1.5 Step 4 for the detailed matching argument.

STEP 2 (GLOBAL HAMILTONIAN VIA SUMMATION AND SMOOTH CONCATENATION).

Following the two-phase assembly of Node 1.5:

Phase A (vertices): H_A = sum_v H_v. All vertex Hamiltonians H_v have disjoint supports (supp(H_v) subset B_delta(v), and the balls are disjoint by Node 1.5 Step 1). Therefore {H_v, H_w} = 0 for v != w (Poisson bracket vanishes for functions with disjoint support). The sum H_A is a well-defined smooth function, and its Hamiltonian flow is the simultaneous application of all vertex smoothings:

  phi_t^{H_A} = composition of phi_t^{H_v} over all v.

Reference: McDuff-Salamon, Introduction to Symplectic Topology, 3rd ed., Proposition 5.4.2.

Phase B (edges): After Phase A, the surface phi_1^{H_A}(K) has smooth vertex regions but singular edges (outside vertex balls). Define H_B = sum_e H_e. The edge Hamiltonians have pairwise disjoint supports (tubular neighborhoods of distinct edge exteriors are disjoint by Node 1.5 Step 2). So {H_e, H_f} = 0 for distinct edges e, f. The sum H_B is well-defined.

PHASE B SUPPORT CLARIFICATION: H_A is supported in the union of vertex balls; H_B is supported in the edge tubular neighborhoods, which are OUTSIDE the vertex balls. Therefore supp(H_A) and supp(H_B) are disjoint. In particular, on supp(H_B), the Phase A flow phi_1^{H_A} acts as the identity (since H_A = 0 there). Consequently, the pullback (phi_1^{H_A})* H_B = H_B circ (phi_1^{H_A})^{-1} equals H_B on supp(H_B) (because (phi_1^{H_A})^{-1} = id on supp(H_B)). So the concatenation formula simplifies: the pullback notation is technically correct but equals the identity transformation on the relevant support.

SMOOTH TIME REPARAMETRIZATION FOR CONCATENATION. The naive concatenation H_t = 2*H_A on [0,1/2] then 2*H_B on [1/2,1] would be discontinuous at t = 1/2 (jumping from 2*H_A to 2*H_B). We use smooth bump reparametrization instead.

Let rho: [0,1] -> [0,1] be a smooth monotone function with:
  rho(t) = 0 for t <= 0,
  rho(t) = 1 for t >= 1,
  rho flat at both endpoints (all derivatives zero at t = 0 and t = 1).

Such rho exists (e.g., rho(t) = exp(-1/t) / [exp(-1/t) + exp(-1/(1-t))] on (0,1)).

Define the time-reparametrized Hamiltonians:
  H_t^{Phase A} = rho'(2t) * H_A,     for t in [0, 1/2],
  H_t^{Phase B} = rho'(2t - 1) * H_B,  for t in [1/2, 1].

The concatenated Hamiltonian is:
  H_t = H_t^{Phase A}   for t in [0, 1/2],
  H_t = H_t^{Phase B}   for t in [1/2, 1].

SMOOTHNESS AT t = 1/2: For t near 1/2 from below: H_t^{Phase A} = rho'(2t) * H_A. Since rho(s) = 1 for s >= 1 (and we have 2t -> 1 as t -> 1/2), rho' and all higher derivatives vanish at s = 1. So H_t^{Phase A} and all its time-derivatives vanish as t -> 1/2^-. For t near 1/2 from above: H_t^{Phase B} = rho'(2t-1) * H_B. Since rho(s) = 0 for s <= 0 (and 2t-1 -> 0 as t -> 1/2), rho' and all higher derivatives vanish at s = 0. So H_t^{Phase B} and all its time-derivatives vanish as t -> 1/2^+. Both pieces and all their time-derivatives are zero at t = 1/2. The concatenation is C^infinity in t.

FLOW VERIFICATION: The Phase A flow integral_0^{1/2} rho'(2t) dt = (1/2) integral_0^1 rho'(s) ds = (1/2)(rho(1) - rho(0)) = 1/2. Wait — we need the flow to complete the full Phase A deformation by t = 1/2. The correct normalization: the flow of rho'(2t)*H_A on [0,1/2] is phi_{rho(2t)}^{H_A}. At t = 1/2: rho(1) = 1, so the flow is phi_1^{H_A}. Similarly, the flow of rho'(2t-1)*H_B on [1/2,1] is phi_{rho(2t-1)}^{H_B}. At t = 1: rho(1) = 1, so the flow is phi_1^{H_B}. The total flow at t = 1 is phi_1^{H_B} circ phi_1^{H_A}, which maps K to L.

This is the standard smooth concatenation of Hamiltonian isotopies; see McDuff-Salamon, 3rd ed., Proposition 5.4.6 (composition of Hamiltonian diffeomorphisms is Hamiltonian).

STEP 3 (COMPLETENESS OF THE HAMILTONIAN FLOW).

Case 1: Compact K. All vertices lie in a bounded region, all edges are finite. Every H_v and H_e is compactly supported. Their sums H_A, H_B are compactly supported. Compactly supported Hamiltonian vector fields on R^4 are complete: the flow exists for all t in R. Reference: McDuff-Salamon, 3rd ed., Section 5.4.

Case 2: Non-compact K with finitely many vertices but infinite edges. All H_v are compactly supported; H_A is compactly supported. For infinite edges, H_e is not compactly supported but: (i) H_e is bounded: |H_e| <= C. (ii) X_{H_e} is bounded: |X_{H_e}| <= C'. (iii) X_{H_e} is tangent to the transverse (x_2, y_2)-plane (H_e is x_1-independent). A bounded Hamiltonian vector field has a complete flow (integral curves cannot escape to infinity in finite time since |dx/dt| <= C'). Reference: Lang, Differential and Riemannian Manifolds, Theorem IV.1.6.

Case 3: Non-compact K with infinitely many vertices. Exhaustion argument. For each interior vertex v of K (a vertex whose star is fully contained in K), the Hamiltonian H_v is determined by the local geometry in B_delta(v). We define H_infinity = sum_v H_v + sum_e H_e as a locally finite sum.

STABILIZATION: For any compact subset S of R^4, only finitely many vertex balls and edge neighborhoods intersect S (by local finiteness of the polyhedral structure). So H_infinity restricted to S equals a finite sum. In particular, H_infinity is smooth (locally equals a finite sum of smooth functions).

BOUNDARY TREATMENT: In the exhaustion K_n = K cap B_n(0), we smooth only INTERIOR vertices (those whose star lies entirely within B_n(0)). Boundary vertices (where faces extend beyond B_n) are NOT smoothed. This avoids any issue with choosing consistent extensions. For n large enough that v's star is inside B_n, the Hamiltonian H_v is the same in K_n and K_{n+1} (it depends only on the local geometry in B_delta(v), which is identical). This gives stabilization without boundary extension issues.

SMOOTHNESS OF H_infinity: By local finiteness plus stabilization, H_infinity is smooth on R^4 (it locally equals H_n for n large enough).

COMPLETENESS: On any compact set S, the flow of H_infinity agrees with the flow of H_n for n large enough (since H_infinity = H_n on a neighborhood of S for large n, and H_n is compactly supported hence complete). The flow on compact sets stabilizes, defining a well-defined smooth flow phi_t^{H_infinity} on all of R^4. This flow is the desired Hamiltonian diffeomorphism smoothing all of K.

STEP 4 (FLUX VANISHING — CORRECT FORMULATION).

The Lagrangian flux of an isotopy L_t is an element of H^1(L_0; R), defined by:
  Flux({L_t})(gamma) = integral_0^1 (integral_gamma omega(V_t, .)) dt,
where V_t = dL_t/dt and gamma is a loop in L_0.

IMPORTANT DISTINCTION: The Lagrangian flux lives in H^1(L_0; R), NOT in H^1(R^4; R). These are different cohomology groups. The vanishing of H^1(R^4; R) = 0 does not directly imply the vanishing of the Lagrangian flux when H^1(L_0; R) != 0.

However, this distinction is MOOT for our construction, because our isotopy is Hamiltonian BY CONSTRUCTION (Steps 1-3 above). Here is why flux vanishes independently:

(a) Construction-based argument (primary): The isotopy K_t = phi_t^H(K) is generated by an ambient Hamiltonian vector field X_H on R^4. The flux integral over any loop gamma becomes integral_gamma dH_t = 0.

(b) Exactness argument (secondary, independent verification): Each K_t in the isotopy is an exact Lagrangian. We verify this region by region:

(b1) NEAR VERTICES (inside B_delta(v)): K_t is the graph of dG_t where G_t is the generating function from Node 1.3. The Liouville form lambda = sum y_i dx_i pulls back to dG_t on graph(dG_t), which is exact. So lambda|_{K_t near v} = dG_t (exact).

(b2) ALONG EDGES (outside vertex balls): K_t = {(x_1, gamma_t(s), 0, gamma_t(s))} in edge-adapted coordinates. The pullback: lambda|_{K_t} = y_1 dx_1 + y_2 dx_2 = 0 * dx_1 + gamma_{t,2}(s) * gamma_{t,1}'(s) ds. Define g_t(s) = integral_0^s gamma_{t,2}(u) * gamma_{t,1}'(u) du. Then lambda|_{K_t along e} = dg_t(s) (exact, since the form depends only on the single parameter s).

(b3) AT JUNCTIONS (where vertex regions meet edge regions): Near the Phase A/Phase B junction, the vertex-region primitive is G_t (generating function) and the edge-region primitive is g_t(s). On the overlap (where both descriptions are valid and the surface is the original PL surface K since Phase B has not yet acted), both primitives restrict to the PL generating function. The difference G_t - g_t is locally constant on the overlap (connected arc of the boundary circle intersected with the face). Since we can always adjust primitives by constants, the local primitives patch to a global primitive on K_t. (Alternatively: K_t is simply connected — it is a topological disk near each vertex, an annulus along each edge — so H^1(K_t; R) = 0 whenever K_t is simply connected, making the patching automatic.)

(b4) FOR GENERAL TOPOLOGY: If K is not simply connected (e.g., a torus), H^1(K; R) != 0. ### Node 1.7

**Statement:** Topological extension to t=0 (Lemma). Depends on: Node 1.3 (vertex smoothing), Node 1.4 (edge smoothing), Node 1.5 (global assembly), Node 1.6 (Hamiltonian isotopy). Inference type: lemma.

STATEMENT. The Hamiltonian isotopy K_t constructed in Nodes 1.3-1.6 extends continuously to t=0 in the Hausdorff topology, with K_0 = K. That is, the map [0,1] -> {closed subsets of R^4} given by t -> K_t is continuous at t=0 when the codomain is equipped with the Hausdorff metric.

PROOF.

The proof proceeds in three parts: (1) vertex generating function convergence as the inner-zone radius shrinks to zero, (2) edge profile convergence as the rounding parameter goes to zero, (3) global Hausdorff distance estimate showing the smooth surfaces converge to the polyhedral surface.

STEP 1 (VERTEX GENERATING FUNCTION CONVERGENCE).

Recall from Node 1.3 (Session 5 rewrite, two-zone construction) that the vertex smoothing uses a family of generating functions parametrized by an inner-zone radius epsilon(t) = (delta/8)*tau(t), where tau: [0,1] -> [0,1] is smooth with tau(0) = 0, tau(1) = 1, and tau flat at t=0 (all derivatives tau^{(k)}(0) = 0 for k >= 1).

For t in (0,1], the smooth generating function is:

  G_t(X) = F^{(epsilon(t))}(X) = chi(|X|/epsilon(t)) * F_angular(X),

where:
- chi: R -> [0,1] is smooth with chi(s) = 0 for s <= 1, chi(s) = 1 for s >= 2, and chi flat at both transitions,
- F_angular(X) = sum_{i=1}^4 rho_i(theta(X)) * (1/2)X^T A_i X is smooth on R^2 \ {0} (the angular partition of unity interpolation between the four face generating functions),
- F_PL is the piecewise-linear generating function (F_PL = F_i = (1/2)X^T A_i X on sector closure(Omega_i)).

The smooth Lagrangian surface is L_t = Graph(dG_t) restricted to B_delta(v).

POINTWISE CONVERGENCE OF G_t TO F_PL. Fix X in R^2 \ {0}.

(a) If X lies in a sector interior Omega_i (away from the sector boundaries, i.e., the coordinate axes): then F_angular(X) = F_i(X) = F_PL(X) (since rho_i = 1 deep inside sector i and F_i equals the restriction of F_PL to that sector).

(b) For t > 0 small enough that epsilon(t) < |X|/2: we have |X|/epsilon(t) > 2, so chi(|X|/epsilon(t)) = 1. Therefore G_t(X) = F_angular(X) = F_PL(X) on sector interiors.

(c) As t -> 0+: epsilon(t) -> 0 (since tau(0) = 0). For any fixed X != 0, eventually epsilon(t) < |X|/2, so chi(|X|/epsilon(t)) = 1 for all t sufficiently small. Thus G_t(X) -> F_angular(X) = F_PL(X).

(d) On sector boundaries (the four coordinate semi-axes): F_angular smoothly interpolates between adjacent face values across the angular transition strips. As epsilon(t) -> 0, chi(|X|/epsilon(t)) -> 1 for all X != 0, so G_t(X) -> F_angular(X). On the boundary between sectors i and j, F_angular equals F_i = F_j (since F_PL is continuous across sector boundaries, and F_angular is a smooth interpolation that matches F_PL at sector interiors and transitions smoothly across boundaries). Therefore G_t(X) -> F_PL(X) on sector boundaries as well.

(e) At X = 0: G_t(0) = chi(0) * F_angular(0) = 0 for all t (since chi(0) = 0 and F_angular(0) = 0). Also F_PL(0) = 0. So G_t(0) = F_PL(0) = 0 for all t.

CONCLUSION: G_t(X) -> F_PL(X) pointwise on all of R^2 as t -> 0+.

STEP 2 (C^0 CONVERGENCE OF VERTEX SMOOTHING SURFACES).

The smooth Lagrangian is L_t = Graph(dG_t) = {(X, dG_t(X)) : X in B_delta(v)} subset R^4.
The polyhedral Lagrangian is K_v = Graph(dF_PL) restricted to the vertex neighborhood.

GRADIENT CONVERGENCE. For X in a sector interior Omega_i with |X| > 2*epsilon(t):

  dG_t(X) = d(F_angular(X)) = d(F_PL(X)),

since G_t = F_angular = F_PL on this region. So dG_t(X) = dF_PL(X) exactly.

For |X| <= 2*epsilon(t) (the inner and transition zones):

  dG_t(X) = d(chi(|X|/epsilon(t)) * F_angular(X))
          = chi'(|X|/epsilon(t)) * (X/|X|) * (1/epsilon(t)) * F_angular(X) + chi(|X|/epsilon(t)) * dF_angular(X).

Since F_angular(X) = O(|X|^2), we have F_angular(X) = O(epsilon(t)^2) on {|X| <= 2*epsilon(t)}. Similarly, dF_angular(X) = O(|X|) = O(epsilon(t)). The cutoff derivative chi' is bounded. Therefore:

  |dG_t(X) - dF_angular(X)| = O(epsilon(t))   on {|X| <= 2*epsilon(t)}.

Since dF_angular is smooth on R^2 \ {0} and F_angular = F_PL on sector interiors, we have (for X on the PL surface near a sector boundary):

  |dG_t(X) - dF_PL(X)| = O(epsilon(t)).

HAUSDORFF DISTANCE ESTIMATE AT A SINGLE VERTEX. The Hausdorff distance between two graphs over the same base is controlled by the sup-norm of the difference of their fiber values. For L_t = Graph(dG_t) and K_v = Graph(dF_PL):

  d_H(L_t cap B_delta(v), K_v cap B_delta(v)) <= sup_{X in B_delta(v)} |dG_t(X) - dF_PL(X)|.

From the analysis above:
- On {|X| > 2*epsilon(t)} cap B_delta(v): dG_t = dF_PL (exactly on sector interiors).
- On {|X| <= 2*epsilon(t)}: |dG_t - dF_PL| = O(epsilon(t)).

Since 2*epsilon(t) = 2*(delta/8)*tau(t) = (delta/4)*tau(t) -> 0 as t -> 0, the region where dG_t differs from dF_PL shrinks to {0}. On this region, the deviation is O(epsilon(t)) -> 0. Therefore:

  d_H(L_t cap B_delta(v), K_v cap B_delta(v)) = O(epsilon(t)) -> 0   as t -> 0.

STEP 3 (EDGE PROFILE CONVERGENCE).

From Node 1.4 (edge smoothing via explicit Lagrangian profile), the edge smoothing replaces the V-shaped transverse profile with a smooth curve gamma_epsilon(s) in the (x_2, y_2)-plane. The parameter epsilon controls the rounding radius near the origin of the V.

NODE 1.4 CONSTANT-PROFILE CONSTRUCTION. In the edge-adapted Darboux coordinates, the polyhedral edge has the V-shaped profile:

  gamma_0(s) = s * r_1   for s >= 0,
  gamma_0(s) = |s| * r_2   for s < 0,

where r_1, r_2 are unit vectors in the (x_2, y_2)-plane (the two face directions).

The smooth profile gamma_epsilon is chosen so that:
- gamma_epsilon(s) = gamma_0(s) for |s| >= S (agrees with the V outside a compact set),
- gamma_epsilon smoothly rounds the corner in {|s| < S}, with the rounding contained in a ball of radius epsilon around the origin.

For the isotopy {gamma_t}_{t in [0,1]}, with gamma_t -> gamma_0 (the V-shape) as t -> 0:

HAUSDORFF CONVERGENCE. The smoothed edge surface is L_t^edge = {(x_1, gamma_t(s), 0, gamma_t(s)) : x_1 in I, s in R}, where I is the edge segment interval (after removing vertex balls).

The polyhedral edge is K_edge = {(x_1, gamma_0(s), 0, gamma_0(s)) : x_1 in I, s in R}.

Since gamma_t converges uniformly to gamma_0 on compact s-intervals (the rounding shrinks to radius 0), we have:

  d_H(L_t^edge, K_edge) <= sup_{s in R} |gamma_t(s) - gamma_0(s)| -> 0   as t -> 0.

STEP 4 (GLOBAL HAUSDORFF CONVERGENCE).

The global smooth surface K_t is assembled from vertex smoothings (Phase A) and edge smoothings (Phase B), as described in Node 1.5. The polyhedral surface K is the union of vertex neighborhoods (PL cones) and edge segments (V-shaped profiles).

VERTEX CONTRIBUTIONS. From Step 2, for each vertex v:

  d_H(K_t cap B_delta(v), K cap B_delta(v)) = O(epsilon(t)) -> 0.

EDGE CONTRIBUTIONS. From Step 3, for each edge e:

  d_H(K_t cap N_epsilon(e^o), K cap N_epsilon(e^o)) -> 0,

where N_epsilon(e^o) is the tubular neighborhood of the edge exterior (outside vertex balls) and the convergence follows from the uniform convergence of the smooth profile to the V-shape.

JUNCTION REGIONS. At the Phase A/Phase B junctions (where vertex balls meet edge tubular neighborhoods), the convergence follows from the fact that:

### Node 1.8

**Statement:** Global obstruction analysis (Verification lemma). Depends on: Node 1.2 (vertex classification), Node 1.3 (cotangent generating function construction). Inference type: lemma.

STATEMENT. Let K be a polyhedral Lagrangian surface in (R^4, omega_std) with exactly 4 faces at every vertex, and let K_t (t in (0,1]) be the smooth Lagrangian isotopy produced by the construction in Nodes 1.3-1.5. Then no topological, symplectic, or Floer-theoretic obstruction prevents the existence of K_t. Specifically: (a) the Maslov class of K_t is well-defined and extends the piecewise-constant Maslov data on K; (b) Floer-theoretic obstructions are absent (for non-compact K by exactness, for compact K by irrelevance to the smoothing method); (c) topological obstructions are absent (compact K must be a torus, non-compact K has no constraint); (d) no monodromy obstruction prevents global consistency of the construction; (e) regularity, embeddedness, energy, and h-principle concerns are addressed by the explicit construction. This node serves as a CONSISTENCY CHECK: it verifies that the smoothing produced by Nodes 1.3-1.5 does not violate any known obstruction to Lagrangian embeddings.

PROOF.

PART (a): MASLOV CLASS — CONTRACTIBILITY ARGUMENT.

The Maslov class mu in H^1(L; Z) of a smooth Lagrangian L in (R^4, omega_std) is the obstruction to trivializing the Lagrangian Gauss map gamma: L -> LG(2,4), where LG(2,4) = U(2)/O(2) is the Lagrangian Grassmannian with pi_1(LG(2,4)) = Z generated by the Maslov class.

For the polyhedral Lagrangian K, the Gauss map is piecewise constant: on each face F_i, the tangent plane is a fixed Lagrangian plane Pi_i in LG(2,4). At a vertex v, the link (a simple closed curve in S^3 consisting of 4 great circle arcs) determines a loop in LG(2,4) by tracing the Lagrangian tangent planes: gamma_v = Pi_1 -> Pi_2 -> Pi_3 -> Pi_4 -> Pi_1 (cyclically).

Definition. The LOCAL MASLOV INDEX at vertex v is mu(v) := [gamma_v] in pi_1(LG(2,4)) = Z, i.e., the winding number of this loop of Lagrangian planes.

Computation via CONTRACTIBILITY OF COTANGENT CHART. By Node 1.3 (Step 2), there exists a Lagrangian plane Lambda transverse to all four tangent planes Pi_1, ..., Pi_4 at each vertex v. In the cotangent graph coordinates determined by Lambda, all four planes are expressed as graphs of symmetric 2x2 matrices:
  Pi_i = graph(A_i),   A_i symmetric,   i = 1, ..., 4.

The set of all Lagrangian planes transverse to Lambda is the COTANGENT CHART:
  U_Lambda = {graph(A) : A a symmetric 2x2 matrix} subset LG(2,4).

This chart is an affine cell in LG(2,4), diffeomorphic to the space of symmetric 2x2 matrices, which is R^3 (parameterized by three entries a_{11}, a_{12}, a_{22}).

Since all four planes Pi_1, ..., Pi_4 lie in U_Lambda = R^3, and R^3 is CONTRACTIBLE, ANY loop through these four points is null-homotopic. In particular:
  mu(v) = [gamma_v] = 0 in pi_1(LG(2,4)).

This argument works regardless of whether consecutive planes are transverse to each other. The non-transversality of consecutive pairs (dim(Pi_i cap Pi_{i+1}) = 1 for adjacent faces sharing an edge) does NOT affect the conclusion: the loop lies in a contractible subset of LG(2,4), period.

REMARK ON THE SIGNATURE FORMULA. For completeness, we note that the explicit matrices in the gamma=0 normal form are A_1 = (1/2)Id, A_2 = diag(1/2,-1/2), A_3 = -(1/2)Id, A_4 = diag(-1/2,1/2). The consecutive differences A_{i+1}-A_i are each singular (rank 1), reflecting the non-transversality of consecutive planes. The signature formula sign(B-A) for the relative Maslov index between graph(A) and graph(B) is valid only when B-A is nonsingular (i.e., the planes are transverse). For non-transverse pairs, one must use the Robbin-Salamon crossing form or the CLM (Cappell-Lee-Miller) index. However, these refinements are unnecessary here: the contractibility argument above establishes mu(v)=0 without computing any pairwise contributions.

For general gamma: By Node 1.2, every vertex is equivalent to gamma=0 via Sp(4,R), and the Maslov index is a symplectic invariant. So mu(v) = 0 for all vertices.

Consequence for the smoothing: Since mu(v) = 0 at every vertex, the piecewise-constant Gauss map on K has trivial winding around every vertex. The smoothed Gauss map (from the two-zone construction in Node 1.3) extends the piecewise-constant map smoothly across vertices without Maslov class obstructions. The resulting smooth Lagrangian K_t has Maslov class mu(K_t) in H^1(K_t; Z) that restricts to 0 on every small loop around a former vertex.

PART (b): FLOER THEORY — ANALYSIS SPLIT BY COMPACTNESS.

The original claim that 'R^4 exact implies K_t exact implies m_0 = 0' is INCORRECT for compact Lagrangians. Corrected analysis:

CASE 1: K non-compact. The Liouville form lambda = (1/2) sum_i (x_i dy_i - y_i dx_i) restricts to K_t. In the cotangent graph construction (Node 1.3), each local piece of K_t is the graph of an exact 1-form dF, so lambda restricts to an exact 1-form locally. If H^1(K_t; R) = 0 (e.g., K_t contractible), then K_t is exact. Even for non-compact K_t with nontrivial H^1, Floer theory for non-compact Lagrangians requires additional structure (wrapping, stops, etc.) and does not produce obstructions to the existence of K_t as a smooth Lagrangian. The smoothing question for non-compact K is purely local-to-global (Nodes 1.3-1.5) and requires no Floer-theoretic input.

CASE 2: K compact (torus). If K is a compact polyhedral Lagrangian torus, K_t is a smooth compact Lagrangian torus. By Gromov (1985), there is NO closed exact Lagrangian in C^n, so K_t is NOT exact. Non-exact tori can bound holomorphic disks (Cho-Oh 2006: Clifford torus has m_0 != 0).

However, this does NOT obstruct the smoothing: (i) The construction in Nodes 1.3-1.5 is PURELY LOCAL: it replaces singular neighborhoods with smooth Lagrangian patches using generating functions and compactly supported Hamiltonians, never invoking Floer theory. (ii) Floer-theoretic obstructions (m_0 != 0, obstructed A-infinity structures) concern the Fukaya-categorical properties of the RESULTING smooth Lagrangian, not the existence of the smoothing. (iii) No Floer-theoretic invariant of K is known to obstruct the existence of a smooth Lagrangian representative.

Conclusion: Floer-theoretic obstructions are IRRELEVANT to the smoothing problem. The smoothing construction is Floer-agnostic.

PART (c): TOPOLOGICAL OBSTRUCTIONS.

COMPACT CASE. A compact Lagrangian surface L in (R^4, omega_std) satisfies chi(L) = 0 (Whitney: normal bundle isomorphic to tangent bundle via J, so the Euler class of the normal bundle equals the Euler class of the tangent bundle, giving 2*chi(L) = chi(L) + chi(L) from Whitney sum TL + NL = TR^4|_L trivial, hence chi(L) = 0).

Closed surfaces with chi = 0 are: the torus T^2 (orientable) and the Klein bottle K^2 (non-orientable).

The Klein bottle case requires careful analysis. A smooth Lagrangian Klein bottle in C^2 does NOT exist, by the independent results of Shevchishin (2009, J. Symplectic Geom. 7(1), 85-106: 'Lagrangian embeddings of the Klein bottle and combinatorial properties of mapping class groups') and Nemirovski (2009, Geom. Funct. Anal. 19(3), 902-909: 'Homology class of a Lagrangian Klein bottle'). Both proved that there is no Lagrangian embedding of the Klein bottle in C^2 (= R^4 with the standard symplectic structure).

CONSEQUENCE FOR THE MAIN THEOREM. There are two logical possibilities for compact K with chi(K) = 0:

(i) COMPACT POLYHEDRAL LAGRANGIAN KLEIN BOTTLES WITH 4-VALENT VERTICES DO NOT EXIST. If no such PL Klein bottle exists in R^4, then the compact Klein bottle case is vacuous and the main theorem holds trivially for this case. Evidence for this: the Shevchishin-Nemirovski obstruction is topological (involving the homology class and the Maslov class), and plausibly extends to the polyhedral category. However, a complete proof of non-existence in the polyhedral category would require separate argument.

(ii) SUCH PL KLEIN BOTTLES EXIST BUT CANNOT BE SMOOTHED. If a compact polyhedral Lagrangian Klein bottle with 4-valent vertices does exist in R^4, then it cannot be smoothed to a smooth Lagrangian Klein bottle (by Shevchishin-Nemirovski). In this case, the main theorem requires a TOPOLOGICAL HYPOTHESIS: either assume K is orientable (for compact K), or restrict to K that admit smooth Lagrangian representatives.

RESOLUTION. The main theorem should be stated with the following understanding: For compact K, the theorem applies to polyhedral Lagrangian TORI (which is the only compact orientable surface with chi=0). If K is a compact polyhedral Klein bottle, it falls outside the scope of the theorem — not because the construction fails, but because the TARGET (smooth Lagrangian Klein bottle in C^2) does not exist. This is not a gap in the proof but a necessary scope restriction dictated by symplectic topology.

In practice, the 4-faces-per-vertex condition combined with the manifold condition and Lagrangian condition strongly constrains the topology. For compact K: chi(K) = 0 forces T^2 or K^2, and K^2 is excluded by Shevchishin-Nemirovski from having any smooth Lagrangian representative.

NON-COMPACT CASE. No topological obstruction: R^4 has trivial topology. For non-compact K, the Shevchishin-Nemirovski theorem does not apply (it requires compactness). A non-compact non-orientable Lagrangian surface (e.g., a Lagrangian Mobius-band-type configuration) could in principle exist in R^4. However, the local smoothing of Nodes 1.3-1.5 preserves local topology: it replaces each vertex neighborhood with a smooth disk and each edge neighborhood with a smooth strip. The global (non-)orientability is inherited from K, and since both orientable and non-orientable smooth non-compact Lagrangian surfaces can exist in R^4, no obstruction arises. If the polyhedral K is non-orientable and non-compact, its smoothing K_t is a smooth non-orientable non-compact Lagrangian surface — this is not obstructed.

PART (d): MONODROMY — CANONICAL CHOICE ELIMINATES DISCRETE AMBIGUITY.

The original claim that the parameter space is contractible ('product of half-lines') is INCORRECT for a tropical-resolution-based construction. The corrected analysis, using the cotangent graph construction of Node 1.3:

The construction requires choosing a REFERENCE LAGRANGIAN PLANE Lambda transverse to all 4 tangent planes at each vertex. The set of valid choices at vertex v is: T(v) = LG(2,4) \ (Sigma(Pi_1) cup ... cup Sigma(Pi_4)), where Sigma(Pi_i) is the codimension-1 Maslov cycle. The complement T(v) is open and dense in the connected 3-manifold LG(2,4).

KEY POINT: Unlike a tropical resolution (which requires a DISCRETE partition of 4 faces into 2 pairs, giving 3 choices with pi_0 = Z/3), the cotangent graph method uses a SINGLE reference plane Lambda that makes ALL 4 faces simultaneously into cotangent graphs. A smooth partition of unity in the angular variable then produces the smooth generating function. The construction is CANONICAL once Lambda is chosen, with NO discrete choices.

T(v) is connected: removing finitely many codimension-1 subvarieties from a connected manifold of dimension >= 2 preserves connectedness. The constraints at different vertices are INDEPENDENT (supports are disjoint by Node 1.5, Phase A).

After fixing Lambda, the remaining parameters (cutoff radius delta > 0, angular width alpha > 0, cutoff function phi) live in products of open intervals — a contractible space. (e1) Regularity. The smoothing does NOT pass through intermediate regularity classes (C^1, C^{1,alpha}). (e2) Embeddedness. The smoothed surface K_t is embedded. The argument proceeds in two parts:

LOCAL EMBEDDEDNESS: Near each vertex v, the smoothed surface L_v = Graph(dF_smooth) is the graph of a smooth map X -> (X, dF_smooth(X)) over the ball B_delta(v). Any graph over an open domain in R^2 is automatically an embedded submanifold of R^4 (the projection to the base is a left inverse of the graph map, ensuring injectivity; the differential has rank 2, ensuring immersion).

GLOBAL EMBEDDEDNESS: The original polyhedral K is embedded by hypothesis. The Hamiltonian flow phi_H^t at each vertex is a smooth diffeomorphism of R^4 (for each t), and a diffeomorphism maps embedded submanifolds to embedded submanifolds. The key point is that the Hamiltonian modifications at different vertices have DISJOINT SUPPORTS (Node 1.5, Phase A ensures the support balls B_delta(v_i) are pairwise disjoint). Therefore the modifications at different vertices do not interact. The composed Hamiltonian isotopy Phi = phi_{H_{v_N}}^1 circ ... circ phi_{H_{v_1}}^1 is a diffeomorphism of R^4 (composition of diffeomorphisms), and K_t = Phi(K) is the image of the embedded surface K under the diffeomorphism Phi, hence K_t is embedded.

To be precise: K_t is NOT a single global graph over R^2 — it is assembled from local graph-Lagrangian patches near each vertex, joined by unmodified polyhedral regions away from the 1-skeleton. Different vertices use different local cotangent coordinates centered at different points. The global embeddedness follows from the diffeomorphism argument, not from any single-chart property.

(e3) Energy bounds. The Hamiltonian H_v at each vertex satisfies ||H_v||_{C^0} = O(delta^2). We derive this explicitly from Node 1.3's two-zone construction.

  ||H_v||_{C^0} = O(delta^2).
